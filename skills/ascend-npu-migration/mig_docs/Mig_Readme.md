## 6. 常见问题（FAQ）

| 现象 | 可能原因 | 处理 |
|------|----------|------|
| 加载 OM 失败 | 运行时与编译 CANN 版本不一致 | 对齐版本或重编 OM |
| 输出与基线不一致 | 预处理或输出头接错 | 对照 Mig_report IO 与 Golden 样本 |
| INT8 精度差 | 校准集或预处理不一致 | 重校或改量化配置 |
