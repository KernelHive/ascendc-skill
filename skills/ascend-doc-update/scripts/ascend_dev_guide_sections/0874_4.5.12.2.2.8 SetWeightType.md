## SetWeightType

### 功能说明

设置权重矩阵Weight的位置、数据格式、数据类型信息，这些信息必须与Kernel侧的设置保持一致。

### 函数原型

```cpp
   void SetWeightType(ConvCommonApi::TPosition pos, ConvCommonApi::ConvFormat format,
ConvCommonApi::ConvDtype dtype)
```

### 参数说明

表4-960 参数说明

### 参数名 输入/输出 描述

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| pos | 输入 | Weight在内存上的位置。当前仅支持TPosition::GM。 |
| format | 输入 | Weight的数据格式。当前仅支持<br>`ConvFormat::FRACTAL_Z_3D。` |
| dtype | 输入 | Weight的数据类型。当前仅支持ConvDtype::FLOAT16或者<br>`ConvDtype::BF16。`<br>返回值说明<br>无<br>约束说明<br>无<br>调用示例<br>`auto ascendcPlatform = platform_ascendc::PlatformAscendCManager::GetInstance();`<br>`ConvBackpropApi::Conv3DBpInputTiling conv3DBpDxTiling(*ascendcPlatform);`<br>`conv3DBpDxTiling.SetWeightType(ConvCommonApi::TPosition::GM,`<br>`ConvCommonApi::ConvFormat::FRACTAL_Z_3D,`<br>`ConvCommonApi::ConvDtype::FLOAT16);` |
