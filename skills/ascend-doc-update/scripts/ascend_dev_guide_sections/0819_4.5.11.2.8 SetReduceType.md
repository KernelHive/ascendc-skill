## SetReduceType

### 功能说明

设置Reduce操作类型，仅对有归约操作的通信任务生效。

### 函数原型

```cpp
uint32_t SetReduceType(uint32_t reduceType, uint8_t dstDataType = 0, uint8_t srcDataType = 0)
```

### 参数说明

表4-926 参数说明

### 参数名 输入/输出 描述

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| reduceTyp | 输入 | 归约操作类型，仅对有归约操作的通信任务生效。uint32_t e 类型，取值详见表4-889参数说明。 |
| dstDataTy | 输入 | 通信任务中输出数据的数据类型。uint8_t类型，该参数的取pe 值范围请参考表4-888。<br>Atlas A3 训练系列产品/Atlas A3 推理系列产品，该参数暂不支持，配置后不生效。<br>Atlas A2 训练系列产品/Atlas A2 推理系列产品，该参数暂不支持，配置后不生效。 |
| srcDataTy | 输入 | 通信任务中输入数据的数据类型。uint8_t类型，该参数的取pe 值范围请参考表4-888。<br>Atlas A3 训练系列产品/Atlas A3 推理系列产品，该参数暂不支持，配置后不生效。<br>针对Atlas A2 训练系列产品/Atlas A2 推理系列产品，该参数暂不支持，配置后不生效。 |

### 返回值说明

- 0表示设置成功。
- 非0表示设置失败。
