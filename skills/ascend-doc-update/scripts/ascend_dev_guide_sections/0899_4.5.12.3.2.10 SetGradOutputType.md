## SetGradOutputType

### 功能说明

设置GradOutput的位置、数据格式、数据类型信息，这些信息必须与Kernel侧的设置保持一致。

### 函数原型

```cpp
   void SetGradOutputType(ConvCommonApi::TPosition pos, ConvCommonApi::ConvFormat format,
ConvCommonApi::ConvDtype dtype)
```

### 参数说明

表4-985 参数说明

### 参数名 输入/输出 描述

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| pos | 输入 | GradOutput在内存上的位置。当前仅支持TPosition::GM。 |
| format | 输入 | GradOutput的数据格式。当前仅支持ConvFormat:NDC1HWC0。 |
