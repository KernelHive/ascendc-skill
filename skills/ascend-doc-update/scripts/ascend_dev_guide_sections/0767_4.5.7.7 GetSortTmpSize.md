## GetSortTmpSize

### 功能说明

获取Sort接口所需的临时空间大小。

### 函数原型

```cpp
   uint32_t GetSortTmpSize(const platform_ascendc::PlatformAscendC &ascendcPlatform, const uint32_t
elemCount, const uint32_t dataTypeSize)
```
