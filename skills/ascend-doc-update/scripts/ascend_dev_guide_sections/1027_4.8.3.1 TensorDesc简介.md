## TensorDesc简介

TensorDesc用于储存ListTensorDesc.GetDesc()中根据index获取对应的Tensor描述信息。

### 原型定义

```cpp
template<class T> class TensorDesc {
   TensorDesc();
   ~TensorDesc();
   void SetShapeAddr(uint64_t* shapePtr);
   uint64_t GetDim();
   uint64_t GetIndex();
   uint64_t GetShape(uint32_t offset);
   T* GetDataPtr();
 GlobalTensor<T> GetDataObj();
     }
```
