## SetMadType

### 功能说明

设置是否使能HF32模式。当前版本暂不支持。

### 函数原型

```cpp
int32_t SetMadType(MatrixMadType madType)
```

### 参数说明

表4-671 参数说明

### 参数名 输入/输出 描述

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| madType | 输入 | 设置Matmul模式。MatrixMadType类型，定义如下。<br>`enum class MatrixMadType : int32_t {`<br>`NORMAL = 0,`<br>`HF32 = 1,`<br>`};`<br>- MatrixMadType::NORMAL：普通模式，即非HF32模<br>式。<br>- MatrixMadType::HF32：使能HF32模式。 |

### 返回值说明

- 1表示设置失败； 0表示设置成功。

### 约束说明

### 无
