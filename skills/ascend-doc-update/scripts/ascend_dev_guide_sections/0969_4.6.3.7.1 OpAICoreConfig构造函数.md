## OpAICoreConfig构造函数

### 功能说明

OpAICoreConfig构造函数。

### 函数原型

```cpp
OpAICoreConfig()
OpAICoreConfig(const char *soc)
```

### 参数说明

### 参数名 输入/输出 描述

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| soc | 输入 | AI处理器型号。 |

### 返回值说明

### 无

### 约束说明

传入soc入参的构造函数会对OpAICoreConfig结构中的部分参数进行初始化，具体的参数和初始化值如下表所示：

### 配置参数 说明 初始化值

DynamicCo 用于标识该算子实现是否支持入图时的静态Shape true mpileStatic 编译。Flag

DynamicFor 标识是否根据4.6.3.6.3 SetOpSelectFormat设置 true matFlag 的函数自动推导算子输入输出支持的dtype和format。

DynamicRa 标识算子是否支持dynamicRank（动态维度）。 true nkSupportF lag

DynamicSh 用于标识该算子是否支持入图时的动态Shape场 true apeSupport 景。Flag

NeedCheck 标识是否在算子融合阶段调用算子参数校验函数 false SupportFla 进行data type与Shape的校验。g

PrecisionRe 此字段用于进行ATC模型转换或者进行网络调测 true duceFlag 时，控制算子的精度模式。无入参的默认构造函数不会初始化上述参数。

AddCustom(const char* name) : OpDef(name)
