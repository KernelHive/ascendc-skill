## SetStartPosition

### 产品支持情况

### 功能说明

设置单核上GradOutput载入数据的起始位置。

### 函数原型

```cpp
__aicore__ inline void SetStartPosition(uint32_t curDinStartIdx, int32_t curHoStartIdx)
```

### 参数说明

### 参数名 输入/输出 描述

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| curDinStartI | 输入 | 当前核D方向起始位置。dx |
| curHoStartId | 输入 | 当前核H方向起始位置。x<br>返回值说明<br>无<br>约束说明<br>无<br>调用示例<br>`gradInput_.SetStartPosition(dinStartIdx_, curHoStartIdx_); // 设置单核上GradOutput载入的起始位置` |
