## SoftmaxFlash Tiling接口

### 功能说明

注意：该接口后续即将废弃，新开发内容不要使用该接口。

用于获取SoftmaxFlash Tiling参数。

### 函数原型

- 获取Kernel接口计算所需最小/最大临时空间的接口
```cpp
  uint32_t GetSoftMaxFlashMaxTmpSize(const ge::Shape& srcShape, const uint32_t dataTypeSize, const
bool isUpdate, const bool isReuseSource)
  uint32_t GetSoftMaxFlashMinTmpSize(const ge::Shape& srcShape, const uint32_t dataTypeSize, const
bool isUpdate, const bool isReuseSource)
```

- Tiling计算接口
- AscendC::optiling命名空间下的计算接口
```cpp
void SoftMaxFlashTilingFunc(const ge::Shape& srcShape, const uint32_t dataTypeSize, const
uint32_t localWorkSpaceSize, optiling::SoftMaxTiling& softmaxFlashTiling, const bool isUpdate =
    false)
```

- AscendC命名空间下的计算接口
```cpp
  void SoftMaxFlashTilingFunc(const ge::Shape& srcShape, const uint32_t dataTypeSize, const
  uint32_t localWorkSpaceSize, AscendC::tiling::SoftMaxTiling& softmaxFlashTiling, const bool
isUpdate = false)
```
