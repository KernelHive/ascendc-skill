## SetBiasType

### 功能说明

设置Bias在内存上的位置、数据格式和数据类型。

### 函数原型

```cpp
   void SetBiasType(const ConvCommonApi::TPosition pos, const ConvCommonApi::ConvFormat format, const
ConvCommonApi::ConvDtype dtype)
```

### 参数说明

### 参数名 输入/输出 描述

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| pos | 输入 | Bias在内存上的位置。当前仅支持<br>`TPosition::GM。` |
| format | 输入 | Bias的数据格式。当前仅支持<br>`ConvFormat::ND。` |
| dtype | 输入 | Bias的数据类型。当前仅支持<br>`ConvDtype::FLOAT16、ConvDtype::FLOAT。`<br>返回值说明<br>无<br>约束说明<br>在调用GetTiling接口前，本接口可选调用。若未调用本接口，默认Bias为<br>`pos=TPosition::GM，format=ConvFormat::ND，dtype=ConvDtype::FLOAT16。`<br>调用示例<br>`// 实例化Conv3D Api`<br>`auto ascendcPlatform = platform_ascendc::PlatformAscendC(context->GetPlatformInfo());`<br>`Conv3dTilingApi::Conv3dTiling conv3dApiTiling(ascendcPlatform);`<br>`conv3dApiTiling.SetBiasType(ConvCommonApi::TPosition::GM, ConvCommonApi::ConvFormat::ND,`<br>`ConvCommonApi::ConvDtype::FLOAT32);` |
