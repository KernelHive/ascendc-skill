## SetOutputType

### 功能说明

设置结果矩阵Output在内存上的位置、数据格式和数据类型。

### 函数原型

```cpp
void SetOutputType(const ConvCommonApi::TPosition pos, const ConvCommonApi::ConvFormat format,
const ConvCommonApi::ConvDtype dtype)
```

### 参数说明

### 参数名 输入/输出 描述

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| pos | 输入 | Output在内存上的位置。当前仅支持<br>`TPosition::CO1。` |
| format | 输入 | Output的数据格式。当前仅支持<br>`ConvFormat::NDC1HWC0。` |
| dtype | 输入 | Output的数据类型。当前仅支持<br>`ConvDtype::FLOAT16、ConvDtype::BF16。`<br>返回值说明<br>无<br>约束说明<br>在调用GetTiling接口前，本接口可选调用。若未调用本接口，默认Bias为<br>`pos=TPosition::CO1，format=ConvFormat::NDC1HWC0，`<br>`dtype=ConvDtype::FLOAT16。`<br>调用示例<br>`// 实例化Conv3D API`<br>`auto ascendcPlatform = platform_ascendc::PlatformAscendC(context->GetPlatformInfo());`<br>`Conv3dTilingApi::Conv3dTiling conv3dApiTiling(ascendcPlatform);`<br>`conv3dApiTiling.SetOutputType(ConvCommonApi::TPosition::CO1,`<br>`ConvCommonApi::ConvFormat::NDC1HWC0, ConvCommonApi::ConvDtype::BF16);` |
