# 11. OP_ATTR_NAMES

## 宏功能

用于封装 AI CPU 算子的属性名字。

## 宏原型

```cpp
OP_ATTR_NAMES ::op::FVector<std::string>
```

## 参数说明

| 参数 | 输入/输出 | 说明 |
|------|-----------|------|
| /    | 输入      | AI CPU 算子的属性名，类型为 `std::string` 的 `FVector`。默认值为空的 `FVector`，即 `OP_ATTR_NAMES()`。 |

## 约束说明

无

## 调用示例

```cpp
// 封装 AI CPU 算子的 3 个属性名为 Tindices，T 和 use_locking
OP_ATTR_NAMES({"Tindices", "T", "use_locking"});
```
