##### SetOriginShape

## 函数功能
向 TensorDesc 中设置 Tensor 的原始 Shape。该 Shape 是指原始网络模型的 Shape。

## 函数原型
```cpp
void SetOriginShape(const Shape &originShape)
```

## 参数说明

| 参数名       | 输入/输出 | 描述                                   |
|--------------|-----------|----------------------------------------|
| originShape  | 输入      | 向 TensorDesc 设置原始的 originShape 对象。 |

## 返回值
无。

## 异常处理
无。

## 约束说明
无。
