# 19. OP_WORKSPACE

## 宏功能

用于封装算子显式指定的 workspace 参数。

## 宏原型

```c
OP_WORKSPACE(x...)
```

## 参数说明

| 参数 | 输入/输出 | 说明 |
|------|------------|------|
| x... | 输入 | 算子的 workspace 参数，表示算子计算过程中需要的 Device 内存。支持 aclTensor 或 aclTensorList 类型。 |

## 约束说明

无。

## 调用示例

```c
// 封装算子的 workspace 参数 a
OP_WORKSPACE(a);
```
