##### 简介

CtInferShapeContext 继承自 InferShapeContext，用于编译期 Shape 推导，提供编译期使用的接口。

该类主要用于资源类算子。资源类算子分为两类：

- 创建资源的算子
- 使用资源的算子

这两类算子通过传递 DT_RESOURCE 句柄来关联资源。句柄信息通过 CtInferShapeContext 提供的接口在编译期获取。
