#### IMPLEMT_VERIFIER

## 函数功能

封装算子的 Verify 函数。

## 函数原型

```c
IMPLEMT_VERIFIER(op_name, func_name)
```

## 约束说明

无。

## 参数说明

| 参数名    | 输入/输出 | 描述                         |
|-----------|-----------|------------------------------|
| op_name   | 输入      | 算子类型。                   |
| func_name | 输入      | Verify 函数名，用户自定义。 |

## 返回值

无。
