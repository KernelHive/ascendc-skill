##### 简介

`OpContextBuilderBase` 是各 ContextBuilder 的基类，用于构造子类 context 中算子信息，包括算子类型、名称、输入输出原型个数、输入输出实例个数、属性等信息。

> **说明**
>
> 不可单独构造 `OpContextBuilderBase` 基类对象，只能通过子类构造。

#include "base/context_builder/op_context_builder_base.h"
