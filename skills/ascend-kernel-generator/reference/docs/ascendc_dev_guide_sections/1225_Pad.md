###### Pad

## 支持的产品型号

- Atlas 训练系列产品
- Atlas A2 训练系列产品 / Atlas A2 推理系列产品
- Atlas A3 训练系列产品 / Atlas A3 推理系列产品
- Atlas 推理系列产品

## 功能说明

将输入 tensor 按照 paddings 的大小对各个维度进行填充，填充值为 0。

## 函数原型

```cpp
const aclTensor* Pad(const aclTensor* self, const aclTensor* paddings,
                     aclOpExecutor* executor)
```

## 参数说明

| 参数      | 输入/输出 | 说明 |
|-----------|-----------|------|
| self      | 输入      | 待填充的输入 tensor。<br>数据类型支持：FLOAT16、FLOAT、INT16、UINT16、INT32、INT64、BFLOAT16、INT8。<br>数据格式支持：ND。<br><br>**说明**：BFLOAT16 和 INT8 适用于如下产品型号：<br>- Atlas A2 训练系列产品 / Atlas A2 推理系列产品<br>- Atlas A3 训练系列产品 / Atlas A3 推理系列产品 |
| paddings  | 输入      | 输入 tensor 每个维度被填充的大小，形状为 `[self.dim, 2]`。<br>数据类型支持：INT32、INT64。<br>数据格式支持：ND。 |
| executor  | 输入      | op 执行器，包含了算子计算流程。 |

## 返回值说明

返回被填充了 0 的 tensor。

## 约束说明

无

## 调用示例

```cpp
// 调用 l0op::Pad 对 self 进行补维
l0op::Pad(self, paddingsTensor, executor);
```
