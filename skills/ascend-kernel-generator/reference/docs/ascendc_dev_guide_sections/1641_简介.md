##### 简介

`TilingData` 用于存储算子 Tiling 的计算结果，即 Tiling 实现根据算子的 shape 等信息确定的数据切分算法相关参数（比如每次搬运的数据块大小，搬运、计算操作的循环次数）。

## 需要包含的头文件

```cpp
#include <tensor_data.h>
```
