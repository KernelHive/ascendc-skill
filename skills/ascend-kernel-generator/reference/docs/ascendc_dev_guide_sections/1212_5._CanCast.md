# 5. CanCast

## 功能说明

判断源数据类型是否可以 cast 到目的数据类型。

## 函数原型

```cpp
bool CanCast(const DataType from, const DataType to)
```

## 参数说明

| 参数 | 输入/输出 | 说明 |
|------|-----------|------|
| from | 输入 | 源数据类型 |
| to   | 输入 | 目的数据类型 |

## 返回值说明

若成功 cast 返回 `true`，否则返回 `false`。

## 约束说明

无

## 调用示例

```cpp
// 判断 dtype 不可以转为 int8 类型时，返回
void Func(const DataType dtype) {
    if (!CanCast(dtype, DT_INT8)) {
        return;
    }
}
```
