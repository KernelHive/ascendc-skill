###### DataType

## 功能说明
定义算子参数数据类型。

## 函数原型
```cpp
OpParamDef &DataType(std::vector<ge::DataType> types)
```

## 参数说明

| 参数   | 输入/输出 | 说明                                                                 |
|--------|------------|----------------------------------------------------------------------|
| types  | 输入       | 算子参数数据类型，ge::DataType请参考15.2.3.58 DataType。             |

## 返回值说明
OpParamDef算子定义，OpParamDef请参考15.1.6.1.4 OpParamDef。

## 约束说明
无
