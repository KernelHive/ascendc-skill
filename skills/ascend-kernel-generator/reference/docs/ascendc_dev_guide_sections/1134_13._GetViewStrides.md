# 13. GetViewStrides

## 功能说明

获取 aclTensor 的 ViewStrides。

## 函数原型

```cpp
FVector<int64_t> GetViewStrides()
```

## 参数说明

无

## 返回值说明

返回一个 `FVector` 对象，里面存放 aclTensor 各个维度的 stride 大小。

## 约束说明

无

## 调用示例

```cpp
// 获取 input 的 View Stride 并依次打印出每一维的 stride 大小
void Func(const aclTensor *input) {
    auto strides = input->GetViewStrides();
    for (int64_t stride : strides) {
        std::cout << stride << std::endl;
    }
}
```
