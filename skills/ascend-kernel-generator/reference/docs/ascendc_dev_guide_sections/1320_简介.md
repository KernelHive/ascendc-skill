##### 简介

本类是一个 POD 类，在一块连续的内存空间中存储描述信息以及实际的数据元素（元素类型由模板参数 `T` 指定）。不支持动态扩容。

本类的描述信息包括：最大可保存的元素个数、当前存放的实际元素数量等。

## 需要包含的头文件

```cpp
#include <continuous_vector.h>
```
