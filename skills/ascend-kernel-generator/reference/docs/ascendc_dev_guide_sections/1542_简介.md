##### 简介

`StorageFormat` 是算子数据存储格式管理类，用于表达张量格式、记录不同格式的转换补维规则。该类主要存储：

- **原始格式**：张量输入时的原始格式；
- **运行时格式**：计算过程中实际使用的格式；
- **补维规则**：记录原始格式向运行时格式转换时，Shape 维度扩展规则。
