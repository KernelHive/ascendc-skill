# 5. GetViewShape

## 功能说明

获取 aclTensor 的 ViewShape。ViewShape 表示 aclTensor 的逻辑 shape。

假设一个 aclTensor 的 StorageShape 和 ViewShape 如下：

- StorageShape 为 `[10, 20]`：表示该 aclTensor 在内存上是按照 `[10, 20]` 排布的。
- ViewShape 为 `[2, 5, 20]`：在算子使用时，表示该 aclTensor 可被视为一块 `[2, 5, 20]` 的数据使用。

## 函数原型

```cpp
gert::Shape GetViewShape()
```

## 参数说明

无

## 返回值说明

返回值为 `gert::Shape`，记录了一组 shape 信息，例如一个三维 shape：`[10, 20, 30]`。

## 约束说明

无

## 调用示例

```cpp
void Func(const aclTensor *input) {
  auto shape = input->GetViewShape();
}
```
