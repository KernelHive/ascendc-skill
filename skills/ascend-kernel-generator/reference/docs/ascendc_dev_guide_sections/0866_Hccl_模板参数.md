###### Hccl 模板参数

## 产品支持情况

| 产品 | 是否支持 |
|------|----------|
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 | √ |
| Atlas 200I/500 A2 推理产品 | × |
| Atlas 推理系列产品AI Core | × |
| Atlas 推理系列产品Vector Core | × |
| Atlas 训练系列产品 | × |

## 功能说明

创建Hccl对象时需要传入模板参数。

## 函数原型

Hccl类定义如下，模板参数说明见表1 Hccl类模板参数说明。

```cpp
template <HcclServerType serverType = HcclServerType::HCCL_SERVER_TYPE_AICPU, const auto &config = DEFAULT_CFG>
class Hccl;
```

## 参数说明

**表 15-872 Hccl 类模板参数说明**

| 参数名称 | 描述 |
|----------|------|
| serverType | 支持的服务端类型。HcclServerType类型，定义如下：<br>对于Atlas A2 训练系列产品/Atlas A2 推理系列产品，当前仅支持 `HCCL_SERVER_TYPE_AICPU`。<br>对于Atlas A3 训练系列产品/Atlas A3 推理系列产品，当前仅支持 `HCCL_SERVER_TYPE_AICPU`。<br><br>```cpp<br>enum HcclServerType {<br>    HCCL_SERVER_TYPE_AICPU = 0,<br>    HCCL_SERVER_TYPE_END // 预留参数，不支持使用<br>}<br>``` |
| config | 用于指定向服务端下发任务的核。HcclServerConfig类型，定义如下，默认值 `DEFAULT_CFG = {CoreType::DEFAULT, 0}`。<br><br>```cpp<br>struct HcclServerConfig {<br>    CoreType type;  // 向服务端下发任务的核的类型<br>    int64_t blockId; // 向服务端下发任务的核的ID<br>};<br>```<br><br>CoreType的定义如下：<br><br>```cpp<br>enum class CoreType: uint8_t {<br>    DEFAULT, // 表示不指定AI Cube核或者AI Vector核<br>    ON_AIV,  // 表示指定为AI Vector核<br>    ON_AIC   // 表示指定为AI Cube核<br>};<br>``` |

## 返回值说明

无

## 约束说明

无

## 调用示例

通过如下传入模板参数config的方式创建Hccl类对象，指定Hccl客户端仅在AI Vector的10号核上发送通信消息给服务端，替代通过调用GetBlockIdx接口的方式指定运行的核。

```cpp
static constexpr HcclServerConfig HCCL_CFG = {CoreType::ON_AIV, 10};
// 选择AICPU作为服务端
Hccl<HcclServerType::HCCL_SERVER_TYPE_AICPU, HCCL_CFG> hccl;
```
