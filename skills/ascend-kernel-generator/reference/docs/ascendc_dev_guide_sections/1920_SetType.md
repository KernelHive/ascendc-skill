##### SetType

## 函数功能
设置 ShapeAndType 类的 DataType。

## 函数原型
```cpp
void SetType(DataType data_type)
```

## 参数说明

| 参数名     | 输入/输出 | 描述                     |
|------------|-----------|--------------------------|
| data_type  | 输入      | 设置的目标 DataType。类型请参见 DataType。 |

## 返回值
无。

## 异常处理
无。

## 约束说明
无。
