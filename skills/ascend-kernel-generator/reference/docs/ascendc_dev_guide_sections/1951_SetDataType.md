##### SetDataType

## 函数功能

向 TensorDesc 中设置 Tensor 的数据类型。

## 函数原型

```cpp
void SetDataType(DataType dt)
```

## 参数说明

| 参数名 | 输入/输出 | 描述 |
|--------|-----------|------|
| dt     | 输入      | 需设置的 TensorDesc 所描述的 Tensor 的数据类型信息。<br>关于 DataType 类型，请参见 DataType。 |

## 返回值

无。

## 异常处理

无。

## 约束说明

无。
