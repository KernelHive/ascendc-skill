##### 简介

RuntimeAttrs 是一个算子属性存取类，用于在算子执行过程中提供对属性的访问能力。该类提供了一系列方法，允许用户根据属性索引获取不同类型的属性值，例如 `int64_t`、`float`、`bool`、`string`、`Tensor` 等类型。在 `ExtendedKernelContext` 及其子类中，可通过 `GetAttrs` 接口获取算子属性，返回值中仅包含在 IR 原型定义中的属性。

## 需要包含的头文件

```cpp
#include <runtime_attrs.h>
```
