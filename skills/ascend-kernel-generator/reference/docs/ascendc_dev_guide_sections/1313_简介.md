##### 简介

本类是一个 POD 类，在一块连续的内存空间中存储数据的描述信息以及实际的数据元素，元素类型为 `ContinuousVector` 结构。不支持动态扩容。

本类的描述信息包括：最大可保存的元素个数、当前存放的实际元素数量等。

## 需要包含的头文件

```cpp
#include <continuous_vector.h>
```

## Public 成员函数

### `void Init(const size_t capacity)`

初始化对象。

### `template<typename T> ContinuousVector *Add(size_t inner_vector_capacity)`

添加元素。

### `const ContinuousVector *Get(const size_t index) const`

获取指定索引处的元素。

### `size_t GetSize() const`

获取当前元素数量。

### `static size_t GetOverHeadLength(const size_t capacity)`

获取描述信息长度。
