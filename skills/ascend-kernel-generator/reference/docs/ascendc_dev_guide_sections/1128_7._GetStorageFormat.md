# 7. GetStorageFormat

## 功能说明

获取 aclTensor 的 StorageFormat。  
StorageFormat 表示 aclTensor 在内存中的排布格式，例如 NCHW、ND 等。

## 函数原型

```cpp
op::Format GetStorageFormat()
```

## 参数说明

无

## 返回值说明

返回一个 `op::Format`（即 `ge::Format`），本身是一个枚举，包含多种不同的 Format，例如 NCHW、ND 等。

## 约束说明

无

## 调用示例

```cpp
void Func(const aclTensor *input) {
    auto format = input->GetStorageFormat();
}
```
