## NPU 域调试调优

## 调试方法

NPU域上板数据打印功能包括DumpTensor、printf两种：

- **DumpTensor**：用于打印指定Tensor的数据
- **printf**：主要用于打印标量和字符串信息

### 支持场景

该功能仅在以下场景支持：

- 通过Kernel直调方式调用算子
- 通过单算子API调用方式调用算子
- 间接调用单算子API(aclnnxxx)接口：Pytorch框架单算子直调的场景

### 使用方法

在算子kernel侧实现代码中需要输出日志信息的地方调用DumpTensor接口或者printf接口打印相关内容。

#### DumpTensor示例

```c
DumpTensor(srcLocal, 5, dataLen);
```

参数说明：
- `srcLocal`：待打印的Tensor
- `5`：用户的自定义附加信息（如当前代码行号）
- `dataLen`：元素个数

DumpTensor接口的使用说明和具体约束请参考15.1.7.1 DumpTensor。

**Dump信息格式：**

- 每个block核的dump信息前会增加对应信息头DumpHead（32字节大小），用于记录核号和资源使用信息
- 每次Dump的Tensor数据前也会添加信息头DumpTensorHead（32字节大小），用于记录Tensor的相关信息

#### printf示例

```c
printf("fmt string %d", 0x123);
```

printf接口的使用说明和具体约束请参考15.1.7.2 printf。

### 注意事项

DumpTensor、printf接口打印功能会对算子实际运行的性能带来一定影响，通常在调测阶段使用。开发者可以按需关闭打印功能。具体方法请参考15.1.7.1 DumpTensor、15.1.7.2 printf。

## 性能采集与分析

基于NPU域算子的调用接口编写的算子程序，通过毕昇编译器编译后生成可执行程序，运行可执行程序，可以完成算子NPU域的运行验证。

使用算子调优工具运行NPU模式下生成的可执行文件从而采集Ascend C算子在AI处理器上执行的性能数据，进行性能精细调优。性能调优工具的具体使用方法请参考《算子开发工具用户指南》。
