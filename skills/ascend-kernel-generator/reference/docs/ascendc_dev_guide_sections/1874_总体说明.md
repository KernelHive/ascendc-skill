##### 总体说明

开发人员完成自定义算子的实现代码后，需要进行适配插件的开发，将基于第三方框架的算子映射成适配昇腾AI处理器的算子，可调用 `REGISTER_CUSTOM_OP` 宏实现算子转换。

在调用 `REGISTER_CUSTOM_OP` 宏时，以 `REGISTER_CUSTOM_OP` 开始，以“.”链接 `FrameworkType`、`OriginOpType`、`ParseParamsFn` 等接口。

例如：

```cpp
REGISTER_CUSTOM_OP("OpType")
.FrameworkType(TENSORFLOW)
.OriginOpType("OriginOpType")
.ParseParamsByOperatorFn(ParseParamFunc)
.ImplyType(ImplyType::TVM);
```
