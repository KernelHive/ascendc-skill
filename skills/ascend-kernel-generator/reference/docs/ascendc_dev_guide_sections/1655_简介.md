##### 简介

本类继承自 ContinuousVector 类，与 ContinuousVector 类不同的是 `MutableData` 和 `GetData` 返回的是指定类型的地址，而不是 `void *`。因此称为 Typed。

## 需要包含的头文件

```cpp
#include <continuous_vector.h>
```

## Public 成员函数

| 函数 | 说明 |
|------|------|
| `T *MutableData()` | 返回指定类型的可变数据地址 |
| `const T *GetData() const` | 返回指定类型的只读数据地址 |
