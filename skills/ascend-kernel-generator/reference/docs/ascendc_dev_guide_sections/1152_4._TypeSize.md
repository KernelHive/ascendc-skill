# 4. TypeSize

## 功能说明

计算输入的数据类型一个元素占用多少 Byte。对于不满一个 Byte 大小的数据类型，会在基础 bit 大小上额外加 1000，例如 `Int4` 将返回 1004，表示一个元素占用 4 bit。

## 函数原型

```c
size_t TypeSize(DataType dataType)
```

## 参数说明

| 参数      | 输入/输出 | 说明                               |
|-----------|-----------|------------------------------------|
| dataType  | 输入      | 输入的数据类型，返回该数据类型单元素大小。 |

## 返回值说明

返回输入数据类型单元素的大小。

## 约束说明

无

## 调用示例

```c
// 获取 dtype 的单个元素 size 大小
void Func(const DataType dtype) {
    size_t size = TypeSize(dtype);
}
```
