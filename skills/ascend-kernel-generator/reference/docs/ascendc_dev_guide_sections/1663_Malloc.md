##### Malloc

## 函数功能

在用户内存池中根据指定 size 大小申请 device 内存。

## 函数原型

```cpp
virtual MemBlock *Malloc(size_t size) = 0
```

## 参数说明

| 参数名 | 输入/输出 | 描述                     |
|--------|-----------|--------------------------|
| size   | 输入      | 指定需要申请内存大小。 |

## 返回值

返回 `MemBlock` 指针。

## 异常处理

无。

## 约束说明

纯虚函数，用户必须实现。
