##### 简介

该类描述了 tensor 的 shape，包含两个信息：`origin_shape` 以及 `storage_shape`。其中：

- `origin_shape` 表示 tensor 的 shape 数学描述
- `storage_shape` 表示 `origin_shape` 实际运行时的 shape 格式

## 需要包含的头文件

```cpp
#include <storage_shape.h>
```

## Public 成员函数

| 函数 | 说明 |
|------|------|
| `StorageShape()` | 默认构造函数 |
| `StorageShape(const std::initializer_list<int64_t> &origin_shape, const std::initializer_list<int64_t> &storage_shape)` | 带参构造函数 |
| `const Shape &GetOriginShape() const` | 获取 origin_shape |
| `const Shape &GetStorageShape() const` | 获取 storage_shape |
| `Shape &MutableOriginShape()` | 获取可修改的 origin_shape |
| `Shape &MutableStorageShape()` | 获取可修改的 storage_shape |
| `bool operator==(const StorageShape &other) const` | 相等比较运算符 |
| `bool operator!=(const StorageShape &other) const` | 不等比较运算符 |
