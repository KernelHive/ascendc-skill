##### SetTargets

## 功能说明

设置 Graph 的结束节点列表。

在该列表中的算子需要被执行到，但它的输出不用返回给用户。

## 函数原型

```cpp
Graph &SetTargets(const std::vector<Operator> &targets)
```

## 参数说明

| 参数名 | 输入/输出 | 说明 |
|--------|-----------|------|
| targets | 输入 | 结束节点列表 |

## 返回值说明

| 参数名 | 类型 | 说明 |
|--------|------|------|
| - | Graph& | 返回调用者本身 |

## 约束说明

无
