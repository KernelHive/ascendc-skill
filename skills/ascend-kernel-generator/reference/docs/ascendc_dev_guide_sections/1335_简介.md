##### 简介

`ExpandDimsType`类基于补维后的形状，描述了补维规则。

补维的含义为：在原有形状的基础上，添加一到多个维度。例如原形状 `[2, 2]` 有两根轴，那么在两根轴中间补两维后的形状为 `[2, 1, 1, 2]`。补维后形状的第 0、3 根轴被称为原始轴，第 1、2 根轴被称为补维轴。

`ExpandDimsType` 类通过“1”和“0”描述补维规则：
- “1”代表当前轴为补维轴
- “0”代表当前轴为原始轴

从左到右依次代表当前形状每根轴的来源。

**表 补维规则描述**

| 补维规则 | 补维前形状 | 补维后形状 |
|----------|-------------|------------|
| 0110     | [2, 2]      | [2, 1, 1, 2] |
| 100      | [2, 3]      | [1, 2, 3] |
| 1000     | [2, 3]      | 补维规则与补维前形状不匹配，规则指定原始轴有 3 根，但原始形状只有 2 根轴，补维报错 |
