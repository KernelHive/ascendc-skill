# 4. GetBlockSize

## 功能说明
获取当前 NPU 平台上一个 data block 内存大小（单位 Byte）。

## 函数原型
```cpp
int64_t GetBlockSize()
```

## 参数说明
无

## 返回值说明
返回 `int64_t`。

## 约束说明
无

## 调用示例
```cpp
// 获取当前 NPU 平台上一个 data block 所占的内存大小
void func() {
    int64_t size = GetCurrentPlatformInfo().GetBlockSize();
}
```
