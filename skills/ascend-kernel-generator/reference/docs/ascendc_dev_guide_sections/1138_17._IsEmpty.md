# 17. IsEmpty

## 功能说明
判断 aclTensor 是否为空 tensor。

## 函数原型
```cpp
bool IsEmpty()
```

## 参数说明
无

## 返回值说明
如果 aclTensor 为空 tensor，返回 `true`，否则返回 `false`。

## 约束说明
无

## 调用示例
```cpp
// 判断 input 为空 tensor 则 return，否则获取它的数据类型
void Func(const aclTensor *input) {
    if (input->IsEmpty()) {
        return;
    }
    op::DataType dataType = input->GetDataType();
}
```
