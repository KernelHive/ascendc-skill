# 13. OP_INPUT

## 宏功能
用于封装算子的输入 `aclTensor` 和 `aclTensorList`。

## 宏原型
```c
OP_INPUT(x...)
```

## 参数说明

| 参数 | 输入/输出 | 说明 |
|------|-----------|------|
| x... | 输入 | 输入的 `aclTensor` 和 `aclTensorList` |

## 约束说明
如果算子包含非 `aclTensor` 且非 `aclTensorList` 的输入参数，需要先调用 `aclOpExecutor::ConvertToTensor` 将参数转换为 `aclTensor`。

## 调用示例
```c
// 封装算子的两个输入参数，self 和 other
OP_INPUT(self, other);
```
