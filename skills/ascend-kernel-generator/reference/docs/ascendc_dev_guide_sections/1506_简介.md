##### 简介

```markdown
Range类用于描述一个对象的上下界，包含两个信息：

```cpp
T *min_;
T *max_;
```

其中：
- `min_` 表示对象下界的指针
- `max_` 表示对象上界的指针

用户可以自行定义上下界的类型。

## 需要包含的头文件

```cpp
#include <range.h>
```

## Public 成员函数

| 函数签名 | 说明 |
|---------|------|
| `Range()` | 默认构造函数 |
| `Range(T *min, T* max) : min_(min), max_(max)` | 构造函数，指定上下界 |
| `explicit Range(T *same_ele) : min_(same_ele), max_(same_ele)` | 构造函数，上下界指向同一元素 |
| `bool operator==(const Range<T>& rht) const` | 相等比较运算符 |
| `void SetMin(T *min)` | 设置下界指针 |
| `void SetMax(T *max)` | 设置上界指针 |
| `const T *GetMin() const` | 获取下界指针（const版本） |
| `T *GetMin()` | 获取下界指针 |
| `const T *GetMax() const` | 获取上界指针（const版本） |
| `T *GetMax()` | 获取上界指针 |
```
