# 3. GetOriginalShape

## 功能说明

获取 aclTensor 的 OriginShape 属性。

OriginShape 表示 aclTensor 在经历 transdata 节点前（如果存在该节点）的原始 shape 信息，即 tensor 的 shape 数学描述。

## 函数原型

```cpp
gert::Shape GetOriginalShape()
```

## 参数说明

无

## 返回值说明

返回值为 `gert::Shape`，记录了一组 shape 信息，例如一个三维 shape：`[10, 20, 30]`。

## 约束说明

无

## 调用示例

```cpp
void Func(const aclTensor *input) {
    auto shape = input->GetOriginalShape();
}
```
