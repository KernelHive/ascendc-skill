# 1. GetStorageShape

## 功能说明

获取 aclTensor 的 StorageShape 属性。

StorageShape 表示 aclTensor 在内存上的实际排布，即 OriginShape 实际运行时的 shape 格式。

## 函数原型

```cpp
gert::Shape GetStorageShape()
```

## 参数说明

无

## 返回值说明

返回值为 `gert::Shape`，记录了一组 shape 信息，例如一个三维 shape：`[10, 20, 30]`。

## 约束说明

无

## 调用示例

```cpp
void Func(const aclTensor *input) {
  auto shape = input->GetStorageShape();
}
```
