# 2. GetPrimaryFormat

## 功能说明

获取根格式，去除关于 C0 的信息，只保留单纯的格式信息。例如，将 `FORMAT_FRACTAL_Z_C04` 转为 `FORMAT_FRACTAL_Z`。

## 函数原型

```c
int32_t GetPrimaryFormat(int32_t format)
```

```c
Format GetPrimaryFormat(Format format)
```

## 参数说明

| 参数   | 输入/输出 | 说明               |
|--------|-----------|--------------------|
| format | 输入      | 原始目标数据格式。 |

## 返回值说明

去除 C0 信息后的格式。

## 约束说明

无

## 调用示例

```c
// 判断当 input 的 storage format 是 fractal z 时，返回
void Func(const aclTensor *input) {
    if (GetPrimaryFormat(input->GetStorageFormat()) == FORMAT_FRACTAL_Z) {
        return;
    }
}
```
