## 功能说明

查询属性是否存在。

## 函数原型

```cpp
bool HasAttr(const AscendString &name)
```

## 参数说明

| 参数名 | 输入/输出 | 说明 |
|--------|-----------|------|
| name | 输入 | 指定的属性名。 |

## 返回值说明

| 参数名 | 类型 | 说明 |
|--------|------|------|
| - | bool | True表示存在，False表示不存在。 |

## 约束说明

无
