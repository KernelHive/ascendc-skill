##### 简介

`InferDataTypeContext` 继承自 `ExtendedKernelContext`，是一个用于数据类型（`DataType`）推导的上下文类。该类的主要作用是在算子数据类型推导过程中，提供获取输入、输出数据类型的接口，并支持设置输出数据类型。

## Public 成员函数

| 函数 | 说明 |
|------|------|
| `ge::DataType GetInputDataType(const size_t index) const` | 获取指定索引的输入数据类型 |
| `ge::DataType GetOptionalInputDataType(const size_t ir_index) const` | 获取指定 IR 索引的可选输入数据类型 |
| `ge::DataType GetDynamicInputDataType(const size_t ir_index, const size_t relative_index) const` | 获取动态输入数据类型 |
| `ge::DataType GetRequiredInputDataType(const size_t ir_index) const` | 获取指定 IR 索引的必需输入数据类型 |
| `ge::DataType GetOutputDataType(const size_t index) const` | 获取指定索引的输出数据类型 |
| `ge::graphStatus SetOutputDataType(const size_t index, const ge::DataType datatype)` | 设置指定索引的输出数据类型 |
