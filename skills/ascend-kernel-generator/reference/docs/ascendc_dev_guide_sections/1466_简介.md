##### 简介

`OpInferShapeContextBuilder` 用于构建 `InferShapeContext`。构造出的 Context 在算子 Shape 推导过程中作为入参，用于获取必要的算子输入输出数据。推导完成后，结果会被写入回上下文中。

## 使用步骤

1. **构造 ContextHolder**  
   调用 `OpInferShapeContextBuilder` 接口，传入算子输入的 Tensor 和输出的 TensorDesc 描述等信息，最终调用 `Build()` 接口构造一个 `ContextHolder<InferShapeContext>` 对象。

2. **获取 InferShapeContext**  
   通过 `ContextHolder` 调用 `GetContext` 接口，获取 `InferShapeContext`。

3. **调用算子 Shape 推导函数**  
   调用算子 Shape 推导函数 `InferShapeKernelFunc`，将 `InferShapeContext` 作为函数入参，完成 Shape 推导，并写入推导的输出结果。

4. **获取推导的输出 Shape**  
   通过 `InferShapeContext` 的 `GetOutputShape` 接口可以获取到推导的输出 Shape。

5. **释放 ContextHolder**  
   根据需要释放 `ContextHolder`，释放完成后，此时由 `Build` 构造出来的 `InferShapeContext` 中的数据指针均无效。

## 说明

该类继承自 `OpContextBuilderBase` 类，在 `Build` 构建 `ContextHolder` 对象之前，需要调用 `OpContextBuilderBase` 的 `OpType`、`OpName`、`IONum` 或 `IOInstanceNum`，以及 `AppendAttr` 接口，分别设置算子的类型、名称、输入输出个数、以及算子的属性。
