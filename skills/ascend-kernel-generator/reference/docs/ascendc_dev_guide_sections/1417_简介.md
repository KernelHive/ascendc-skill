##### 简介

InferShapeRangeContext 继承自 ExtendedKernelContext，主要用于获取算子的输入输出 ShapeRange 和输入 TensorRange，其中 ShapeRange 和 TensorRange 分别是 `Range<T>` 模板的特化类型，记录了算子 Shape 和 Tensor 值的上下限。部分算子（例如，NonZero 算子）在进行 shape 推导时，无法推导出具体的输出 shape，只能通过 ShapeRange 信息预估最大 shape。

## Public 成员函数

| 函数 | 说明 |
|------|------|
| `const Range<Shape> *GetInputShapeRange(const size_t index) const` | 获取输入 ShapeRange |
| `const TensorRange *GetInputTensorRange(const size_t index) const` | 获取输入 TensorRange |
| `const TensorRange *GetOptionalInputTensorRange(const size_t ir_index) const` | 获取可选输入 TensorRange |
| `const TensorRange *GetDynamicInputTensorRange(const size_t ir_index, const size_t relative_index) const` | 获取动态输入 TensorRange |
| `const Range<Shape> *GetOptionalInputShapeRange(const size_t ir_index) const` | 获取可选输入 ShapeRange |
| `const Range<Shape> *GetDynamicInputShapeRange(const size_t ir_index, const size_t relative_index) const` | 获取动态输入 ShapeRange |
| `const Range<Shape> *GetRequiredInputShapeRange(const size_t ir_index) const` | 获取必需输入 ShapeRange |
| `const TensorRange *GetRequiredInputTensorRange(const size_t ir_index) const` | 获取必需输入 TensorRange |
| `Range<Shape> *GetOutputShapeRange(const size_t index)` | 获取输出 ShapeRange |
