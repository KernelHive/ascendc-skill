# 3. CREATE_EXECUTOR

## 宏功能

创建一个 UniqueExecutor 对象，该对象为 aclOpExecutor 的生成工厂类。

## 宏原型

```cpp
CREATE_EXECUTOR()
```

## 参数说明

无

## 约束说明

- 在使用 aclOpExecutor 前，必须要调用此宏创建。
- L2 一阶段接口返回前，需要调用 ReleaseTo 将 executor 转移到 L2 接口入参的 executor 中。

## 调用示例

```cpp
// 创建 executor
auto uniqueExecutor = CREATE_EXECUTOR();
```
