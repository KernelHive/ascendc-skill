# 16. GetDataType

## 功能说明

获取 aclTensor 的数据类型。

## 函数原型

```cpp
op::DataType GetDataType()
```

## 参数说明

无

## 返回值说明

`op::DataType`（即 `ge::DataType`）表示 aclTensor 中元素的数据类型，包含多种数据类型，例如 float16、float32 等。

## 约束说明

无

## 调用示例

```cpp
// 获取 input 的数据类型
void Func(const aclTensor *input) {
    op::DataType dataType = input->GetDataType();
}
```
