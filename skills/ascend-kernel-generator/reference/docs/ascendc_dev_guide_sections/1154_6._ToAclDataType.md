# 6. ToAclDataType

## 功能说明

将 `op::DataType` 转换为对应的 `aclDataType`。

- **aclDataType**：该数据类型的详细定义参见《应用开发指南 (C&C++)》中“acl API参考 > 数据类型及其操作接口 > aclDataType”。
- **op::DataType**：算子框架内部定义的数据类型，本质使用的是 GE 定义的 `ge::DataType`。

## 函数原型

```cpp
aclDataType ToAclDataType(DataType type)
```

## 参数说明

| 参数 | 输入/输出 | 说明 |
|------|------------|------|
| type | 输入 | 待转换的原始数据类型 `op::DataType` |

## 返回值说明

返回 `aclDataType` 类型。

## 约束说明

无

## 调用示例

```cpp
// 获取 DT_FLOAT 对应的 aclDataType 枚举
void Func() {
    aclDataType type = ToAclDataType(DT_FLOAT);
}
```
