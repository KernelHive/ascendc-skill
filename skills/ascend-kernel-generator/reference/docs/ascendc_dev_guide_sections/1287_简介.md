##### 简介

## 概述

`ComputeNodeInfo` 用于序列化保存算子的相关编译信息，包括：

- 算子类型
- 算子名称
- 输入输出数量
- 编译时张量的基本信息（`CompileTimeTensorDesc`）
- 输入输出实例化信息（`AnchorInstanceInfo`）

此外，该类还支持获取和设置算子原型中定义的属性的值（`RuntimeAttrs`）。

`ComputeNodeInfo` 的序列化内存空间是平铺式的，内存排布顺序为：

1. `ComputeNodeInfo` 数据成员
2. 算子输入输出实例化信息
3. 编译时张量的基本信息（数量与实际输入、输出个数相同）
4. 算子属性信息
