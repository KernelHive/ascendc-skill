##### GetInputsSize

## 功能说明

返回节点的有效输入个数，即算子的实际输入个数。

## 函数原型

```cpp
size_t GetInputsSize() const
```

## 参数说明

无

## 返回值说明

| 参数名 | 类型   | 说明           |
|--------|--------|----------------|
| -      | size_t | 返回输入个数。 |
