# 1. IsContiguous

## 功能说明
判断输入 tensor 是否为一个连续 tensor。

## 函数原型
```cpp
bool IsContiguous(const aclTensor *tensor)
```

## 参数说明
| 参数   | 输入/输出 | 说明         |
|--------|-----------|--------------|
| tensor | 输入      | 目标 tensor。 |

## 返回值说明
若为连续 tensor 返回 `true`，否则返回 `false`。

## 约束说明
- 传入 tensor 非空。
- 对于空指针场景，将返回 `true`，输出 ERROR 提示：`"check tensor != nullptr failed"`。

## 调用示例
```cpp
// 判断当 tensor 非连续时，返回。
void Func(aclTensor *tensor) {
    if (!IsContiguous(tensor)) {
        return;
    }
}
```
