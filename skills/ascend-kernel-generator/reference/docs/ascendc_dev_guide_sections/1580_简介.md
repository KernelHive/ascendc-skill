##### 简介

Tensor 是张量管理类，用于存储和操作张量数据。该类记录了张量的 shape、数据格式、数据类型、存放位置，并提供了数据地址访问、数据大小计算等接口。

## 需要包含的头文件

```cpp
#include <tensor.h>
```
