# 8. SetStorageFormat

## 功能说明

设置 aclTensor 的 StorageFormat。  
StorageFormat 表示 aclTensor 在内存中的排布格式，例如 NCHW、ND 等。

## 函数原型

```cpp
void SetStorageFormat(op::Format format)
```

## 参数说明

| 参数   | 输入/输出 | 说明                                                                 |
|--------|------------|----------------------------------------------------------------------|
| format | 输入       | 数据类型为 `op::Format`（即 `ge::Format`），本身是一个枚举，包含多种不同的 Format，例如 NCHW、ND 等。 |

## 返回值说明

无

## 约束说明

无

## 调用示例

```cpp
// 将 input 的 Storage Format 置为 ND 格式
void Func(const aclTensor *input) {
    input->SetStorageFormat(ge::FORMAT_ND);
}
```
