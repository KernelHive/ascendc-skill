# 3. AllocIntArray

## 功能说明

申请一个 `aclIntArray`，并使用指定内存中的值对其进行初始化。

## 函数原型

```c
aclIntArray *AllocIntArray(const int64_t *value, uint64_t size)
```

## 参数说明

| 参数 | 输入/输出 | 说明 |
|------|-----------|------|
| value | 输入 | 源数据，用于初始化 `aclIntArray` |
| size | 输入 | 源数据的元素个数 |

## 返回值说明

返回申请到的 `aclIntArray` 对象，申请失败返回 `nullptr`。

## 约束说明

入参指针不能为空。

## 调用示例

```c
// 申请一个长度为10的aclIntArray
void Func(aclOpExecutor *executor) {
    int64_t myArray[10];
    aclIntArray *array = executor->AllocIntArray(myArray, 10);
}
```
