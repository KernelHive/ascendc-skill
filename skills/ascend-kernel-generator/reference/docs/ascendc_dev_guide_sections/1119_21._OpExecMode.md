# 21. OpExecMode

```cpp
enum class OpExecMode : uint32_t {
    // ExecMode support OR operation
    OP_EXEC_MODE_DEFAULT = 0,
    OP_EXEC_MODE_HF32T = 1,
    OP_EXEC_MODE_RESERVED = 0xFFFFFFFF
};
```

`OpExecMode` 类表示算子的运行模式，具体定义如下：

- `OP_EXEC_MODE_DEFAULT = 0` - 默认运行模式
- `OP_EXEC_MODE_HF32T = 1` - HF32T 运行模式
- `OP_EXEC_MODE_RESERVED = 0xFFFFFFFF` - 保留模式

**说明：**
- 运行模式支持 OR 操作
- 使用 `uint32_t` 作为底层类型
