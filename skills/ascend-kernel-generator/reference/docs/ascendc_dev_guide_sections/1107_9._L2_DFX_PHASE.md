# 9. L2_DFX_PHASE_

## 宏功能

用于 L2 二阶段接口 aclnnXxx 时延统计，必须在二阶段接口最前方调用。

## 宏原型

```c
L2_DFX_PHASE_2(APIName)
```

## 参数说明

| 参数     | 输入/输出 | 说明                         |
|----------|-----------|------------------------------|
| APIName  | 输入      | Host 侧 L2 接口名，如 aclnnXxx |

## 约束说明

必须在 L2 二阶段接口的入口处调用，否则可能导致时延统计出现误差。

## 调用示例

```c
// abs 算子的 L2 接口二阶段时延统计
L2_DFX_PHASE_2(aclnnAbs);
```
