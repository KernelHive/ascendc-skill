### 尽量一次搬运较大的数据块

**【优先级】** 高

**【描述】**  
搬运不同大小的数据块时，对带宽的利用率（有效带宽/理论带宽）不一样。  
根据实测经验，单次搬运数据长度在 16KB 以上时，通常能较好地发挥出带宽的最佳性能。因此对于单次搬运，应考虑尽可能搬运较大的数据块。
