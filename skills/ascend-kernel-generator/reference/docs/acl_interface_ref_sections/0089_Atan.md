### Atan

## 输入

### ori_y

- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：反正切函数的y轴坐标值
- **规格限制**：NCHW，支持维度为[1,16,36,90]

### ori_x

- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：反正切函数的x轴坐标值
- **规格限制**：NCHW，支持维度为[1,16,36,90]

## 输出

### ori_pred

- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：y/x的反正切值
- **规格限制**：NCHW，支持维度为[1,16,36,90]
