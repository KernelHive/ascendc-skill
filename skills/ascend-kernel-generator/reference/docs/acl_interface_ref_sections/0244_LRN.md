### LRN

## 功能
对输入 Tensor 局部输入区域进行归一化，局部区域是跨通道定义的。对于 shape 为 `(N x C x D1 x D2, ..., Dk)` 的 Tensor `x[n, c, d1, ..., dk]`，其区域定义为：

## 输入
- **x**：输入 Tensor，数据类型支持 float16、float。

## 属性
- **alpha**：数据类型为 float，缩放因子。
- **beta**：数据类型为 float，指数项。
- **bias**：数据类型为 float，一个可选的 float32。偏移量，通常 >0 以避免除以 0。
- **size**：数据类型为 int，求和的通道数，只支持奇数。

## 输出
- **y**：输出 Tensor，数据类型和 shape 与输入一致。

## 约束与限制
无。

## 支持的 ONNX 版本
Opset v8/v9/v10/v11/v12/v13/v14/v15/v16/v17/v18
