### BatchNorm

## 输入

### x
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输入Tensor
- **规格限制**: 无

### mean
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 均值
- **规格限制**: 无

### variance
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 方差
- **规格限制**: 无

### scale_factor
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 滑动平均的衰减系数，用于前一次计算的mean和variance的影响（除以此值）
- **规格限制**: 无

### scale
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: 接口保留
- **规格限制**: 无

### offset
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: 接口保留
- **规格限制**: 无

## 属性

### eps
- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 分母附加值，防止除以方差时出现除0操作，默认为1e-5
- **规格限制**: 支持配置

### use_global_stats
- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 是否使用mean variance参数，固定取值为True
- **规格限制**: 支持True

## 输出

### y
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: 输出Tensor
- **规格限制**: float16
