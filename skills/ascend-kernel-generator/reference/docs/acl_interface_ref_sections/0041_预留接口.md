## 预留接口

本章罗列了预留的公共接口，后续版本存在变更或废弃的情况，不建议开发者使用，开发者无需关注。

**表 3-1 预留接口表**

| 接口名称 | 接口说明 |
|----------|----------|
| AclSetInputTensorAddr | 功能同 3.34 aclSetInputTensorAddr 一致。 |
| AclSetOutputTensorAddr | 功能同 3.35 aclSetOutputTensorAddr 一致。 |
| AclSetDynamicInputTensorAddr | 功能同 3.31 aclSetDynamicInputTensorAddr 一致。 |
| AclSetDynamicOutputTensorAddr | 功能同 3.32 aclSetDynamicOutputTensorAddr 一致。 |
| AclSetTensorAddr | 功能同 3.37 aclSetTensorAddr 一致。 |
| AclSetDynamicTensorAddr | 功能同 3.33 aclSetDynamicTensorAddr 一致。 |
