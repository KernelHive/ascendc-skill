### FlownetResample

## 输入

### data_in

- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：输入Tensor
- **规格限制**：仅支持输入shape为（1, 2, 96, 128）

## 属性

### add_num

- **是否必填**：非必填
- **数据类型**：float
- **参数解释**：任意值
- **规格限制**：无

### h_proportion

- **是否必填**：非必填
- **数据类型**：float
- **参数解释**：输入的feature map在高方向上的缩放比例
- **规格限制**：仅支持缩放比例为4

### w_proportion

- **是否必填**：非必填
- **数据类型**：float
- **参数解释**：输入的feature map在宽方向上的缩放比例
- **规格限制**：仅支持缩放比例为4

## 输出

### data_out

- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：输出Tensor
- **规格限制**：NC1HWC0
