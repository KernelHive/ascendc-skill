### Cumsum

## 功能
计算输入 Tensor 在给定轴上的累加和。沿给定轴对输入元素执行累积求和。

## 输入
- **x**：输入 Tensor，数据类型支持 float16、float、int32。
- **axis**：一个 0 维向量，指定轴，数据类型支持 int32、int64，范围为 `[-r, r-1]`，默认为 0。负值表示从后面计算维度，`r` 是输入数据 `x` 的维数。

## 属性
- **exclusive**：数据类型为 int，默认为 0，表示返回包括顶层元素的和；如果设置为 1，表示不包括顶层元素的和。
- **reverse**：数据类型为 int，默认为 0，表示正向求和；如果设置为 1，表示反方向求和。

## 输出
- **y**：输出 Tensor，数据类型与输入一致。

## 约束与限制
无。

## 支持的 ONNX 版本
Opset v8/v9/v10/v11/v12/v13/v14/v15/v16/v17/v18
