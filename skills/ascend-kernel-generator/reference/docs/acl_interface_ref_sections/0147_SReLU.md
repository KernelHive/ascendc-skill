### SReLU

## 输入

### x
- **是否必填**：必填
- **数据类型**：float32
- **参数解释**：要进行分段线性激活的输入张量
- **规格限制**：NC1HWC0

### tr
- **是否必填**：必填
- **数据类型**：float32
- **参数解释**：filter权值数据，算子实际使用的是权值文件里面的值，单纯测试单算子，当成输入
- **规格限制**：NC1HWC0，c通道数和输入张量x的c通道数一致，且shape固定为[1,c,1,1]

## 属性

### channel_shared
- **是否必填**：必填
- **数据类型**：bool
- **参数解释**：通道是否共享权值
- **规格限制**：仅支持false
