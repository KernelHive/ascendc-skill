### Exp

## 输入

**x**

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 指数运算，计算公式：
  - 如果 base > 0：`y = base ^ (shift + scale * x)`
  - 如果 base == -1：`y = exp(shift + scale * x)`
- **规格限制**: Float16

## 属性

**base**

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为 -1.0，底数
- **规格限制**: base = -1 或者大于 0

**scale**

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为 1.0
- **规格限制**: 无

**shift**

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为 0
- **规格限制**: 无

## 输出

**y**

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 指数运算，计算公式：
  - 如果 base > 0：`y = base ^ (shift + scale * x)`
  - 如果 base == -1：`y = exp(shift + scale * x)`
- **规格限制**: float16
