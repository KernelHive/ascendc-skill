### Abs

## 功能
对输入 Tensor 取绝对值，其中绝对值 `y = abs(x)` 应用于 Tensor 元素。

## 输入
- **x**：输入 Tensor
  - 数据类型支持：float、float16、int8、int16、int32、int64
  - 数据格式支持：ND

## 输出
- **y**：输出 Tensor
  - 数据类型和 Shape 与输入一致
  - 数据格式支持：ND

## 约束与限制
如果计算量过大可能会导致算子执行超时（aicore error 类型报错，errorStr 为：`timeout or trap error`），场景为最后 2 轴合轴小于 16，前面的轴合轴超大。

## 支持的 ONNX 版本
Opset v8/v9/v10/v11/v12/v13/v14/v15/v16/v17/v18
