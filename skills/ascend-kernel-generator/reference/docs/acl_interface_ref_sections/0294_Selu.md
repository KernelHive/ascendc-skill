### Selu

## 功能

在元素级别使用指数线性单位函数 Selu 对 Tensor 进行计算，并输出结果 Tensor。

## 输入

- **x**：输入 Tensor。数据类型为 int32、int8、float16、float。

## 输出

- **y**：与输入类型相同的 Tensor，且维度与 x 相同。

## 属性

- **alpha**：乘数因子。数据类型为 float。
- **gamma**：乘数因子。数据类型为 float。

## 约束

alpha 或 gamma 只能使用默认值：

- `ALPHA_DEFAULT = 1.67326324235`
- `GAMMA_DEFAULT = 1.05070098736`

## 支持的 ONNX 版本

Opset v8/v9/v10/v11/v12/v13/v14/v15/v16/v17/v18
