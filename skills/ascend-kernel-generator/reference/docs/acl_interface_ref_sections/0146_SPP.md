### SPP

## 输入

### x

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 根据输入的 feature map 确定 pool 的参数，依次输出 1×1，2×2，4×4，… 等平面大小的平面，多个 pool 结果依次排布
- **规格限制**: float16，H 和 W 最大不能超过 510

## 属性

### pyramid_height

- **是否必填**: 必填
- **数据类型**: int
- **参数解释**: 金字塔的层数，其取值范围为 [1,7)，且计算出来的 Pooling 的 kernel_size 和 Pad_size 需要满足 caffe Pooling 的约束（即 Pad_size < kernel_size）
- **规格限制**: [1,7)

### pool_method

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**:
  - 0: max pooling，默认为 max pooling
  - 1: avg pooling
- **规格限制**: 0,1

## 输出

### y

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 根据输入的 feature map 确定 pool 的参数，依次输出 1×1，2×2，4×4，… 等平面大小的平面，多个 pool 结果依次排布
- **规格限制**: float16
