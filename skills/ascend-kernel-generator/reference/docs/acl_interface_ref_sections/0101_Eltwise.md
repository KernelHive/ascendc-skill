### Eltwise

## 动态输入

**x:**
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输入Tensor，Tensor的数据类型和shape必须相同，其Tensor的size最大支持32
- **规格限制**: 输入x的个数 <= 32

## 属性

**operation:**
- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 逐点运算，支持加法、乘法或求最大值操作，取值如下：
  - 0: product
  - 1: sum（默认值为1）
  - 2: max
- **规格限制**: 支持配置范围 0~2

**coeff:**
- **是否必填**: 非必填
- **数据类型**: ListFloat
- **参数解释**: 如果输入，则长度必须等于bottom num。默认为空，在sum时乘以对应的系数，默认值为 `{}`
- **规格限制**: 支持，配置时其向量长度必须等于bottom num

## 输出

**y:**
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输出Tensor，与输入Tensor的数据类型和shape相同，对输入Tensor做逐点运算，支持加法、乘法或求最大值操作
- **规格限制**: 支持float16
