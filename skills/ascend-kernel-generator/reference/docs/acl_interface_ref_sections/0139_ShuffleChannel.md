### ShuffleChannel

## 输入

### x

- **是否必填**: 必填
- **数据类型**: float16、float32、int8、uint8、int16、uint16、int32、uint32、int64、uint64
- **参数解释**: -
- **规格限制**: 无

## 属性

### group

- **是否必填**: 非必填
- **数据类型**: uint
- **参数解释**: 必须大于0，且channel必须能被group整除，默认为1
- **规格限制**: 必须大于0，且channel必须能被group整除

## 输出

### y

- **是否必填**: 必填
- **数据类型**: float16、float32、int8、uint8、int16、uint16、int32、uint32、int64、uint64
- **参数解释**: -
- **规格限制**: 无
