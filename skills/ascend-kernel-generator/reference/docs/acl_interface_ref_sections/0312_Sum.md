### Sum

## 功能
对输入的所有 Tensor 执行逐元素的求和。

## 输入
- **data_i**：Tensor 列表，一组用于求和的 Tensor。  
  数据类型支持：int8、int32、uint8、float16、float。

## 输出
- **sum**：输出 Tensor，数据类型和 shape 与 `data_i` 广播后的 shape 保持一致。

## 限制与约束
- **data_i**：所有输入 Tensor 的数据类型需要保持一致。

## 支持的 ONNX 版本
Opset v8、v9、v10、v11、v12、v13、v14、v15、v16、v17、v18
