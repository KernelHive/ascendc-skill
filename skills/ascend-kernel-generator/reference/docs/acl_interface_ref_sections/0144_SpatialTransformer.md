### SpatialTransformer

## 输入

### x
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: NCHW
- **规格限制**: NCHW

### theta
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: input_theta shape是 (n, size)，其中的size等于theta_1_1-theta_2_3中没有取值的数量，如theta_1_2=1，theta_2_1=2，theta_2_2=3，theta_2_3=4，input_theta=[9,8]，则最终的theta是: 9,1,8,2,3,4
- **规格限制**: ND
