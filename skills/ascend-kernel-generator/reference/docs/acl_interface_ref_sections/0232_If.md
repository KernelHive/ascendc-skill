### If

## 功能
条件分支运算符，基于一个布尔条件，选择性地执行两个不同的子图（`then_branch` 和 `else_branch`）中的一个，并将所选子图的输出传递到外部作用域。

## 输入
- **cond**：条件判断的输入。该 Tensor 必须包含一个元素。数据类型支持 `bool`。

## 属性
- **else_branch**：条件为假的分支。
- **then_branch**：条件为真的分支。

## 输出
- **y**：传递到外部作用域的值，Tensor 或者 Tensor 序列。

## 约束与限制
无。

## 支持的 ONNX 版本
Opset v8/v9/v10/v11/v12/v13/v14/v15/v16/v17/v18
