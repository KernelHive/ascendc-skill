### RNN

## 输入

### x
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: 时间变化数据x，(T×N×...)
- **规格限制**: T最大支持256

### cont
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: 序列连续性标记cont，(T×N)
- **规格限制**: T最大支持256

### x_static（可选输入）
- **是否必填**: 非必填
- **数据类型**: float16
- **参数解释**: 时间不变的静态数据x_static，(N×...)
- **规格限制**: 无

### h_0（可选输入）
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: 初始隐藏状态 (1×N×num_output)
- **规格限制**: 无

### w_xh
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: xh权重，维度为[input_size,num_output]
- **规格限制**: 无

### bias_h
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 偏置，维度为[num_output,]
- **规格限制**: 无

### w_sh
- **是否必填**: 非必填
- **数据类型**: float16
- **参数解释**: sh权重，维度为[input_size,num_output]
- **规格限制**: 无

### w_hh
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: hh权重，维度为[num_output,num_output]
- **规格限制**: 无

### w_ho
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: ho权重，维度为[num_output,num_output]
- **规格限制**: 无

### bias_o
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 偏置，维度为[num_output,]
- **规格限制**: 无

## 属性

### expose_hidden
- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 是否暴露隐藏状态，默认false
- **规格限制**: 无

### num_output
- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 输出特征数，如果为0，则在shape中取值
- **规格限制**: 无

## 输出

### o
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输出的Tensor (T×N×num_output)
- **规格限制**: 无

### h_t
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 最后timestep的隐藏状态 (1×N×num_output)
- **规格限制**: 无
