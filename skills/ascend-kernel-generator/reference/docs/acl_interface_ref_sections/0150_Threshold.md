### Threshold

## 输入

**x:**

- **是否必填:** 必填
- **数据类型:** float16、float32
- **参数解释:** 输入与一个阈值相比较，大于则输出1，小于等于则输出0
- **规格限制:** float16

## 属性

**threshold:**

- **是否必填:** 非必填
- **数据类型:** float
- **参数解释:** 默认为0，输出比较的结果，1或0
- **规格限制:** 无

## 输出

**y:**

- **是否必填:** 必填
- **数据类型:** float16、float32
- **参数解释:** 输入与一个阈值相比较，大于则输出1，小于等于则输出0
- **规格限制:** float16
