### Elu

## 功能

对输入 Tensor `x` 中的每个元素调用指数线性单元激活函数 ELU，并将得到的结果存入输出张量 `y` 中。

## 输入

- **x**：输入 Tensor
  - 数据类型：float16、float

## 属性

- **alpha**：系数
  - 数据类型：float
  - 默认值：1.0

## 输出

- **y**：输出 Tensor
  - 数据类型：与输入 `x` 相同
  - 形状：与输入 `x` 相同

## 约束与限制

无

## 支持的 ONNX 版本

Opset v8/v9/v10/v11/v12/v13/v14/v15/v16
