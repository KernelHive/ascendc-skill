### ConstantOfShape

## 功能
生成具有给定值和形状的张量。

## 输入
- **x**：一维张量，用于表示输出数据的形状。所有值必须大于 0，数据类型支持 int64。

## 属性
- **value**（可选）：指定输出张量的数据和类型，应为单元素张量。如果未指定，则默认为值为 0 且数据类型为 float32 的张量。

## 输出
- **y**：输出张量，由 `x` 指定形状。如果指定了属性 `value`，则输出张量的值和数据类型取自 `value` 指定的值；如果未指定属性 `value`，则输出中的值默认为 0，数据类型默认为 float32。

## 约束
无。

## 支持的 ONNX 版本
Opset v8/v9/v10/v11/v12/v13/v14/v15/v16
