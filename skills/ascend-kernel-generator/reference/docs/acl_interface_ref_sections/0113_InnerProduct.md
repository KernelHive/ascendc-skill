### InnerProduct

## 输入

### x
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: 输入Tensor
- **规格限制**: 无

### w
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: weight的Tensor
- **规格限制**: 无

### b（可选输入）
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: bias
- **规格限制**: 无

## 属性

### num_output
- **是否必填**: 必填
- **数据类型**: int
- **参数解释**: InnerProduct后的神经元数量
- **规格限制**: 最大支持32768

### transpose
- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 是否转置，默认False
- **规格限制**: True或者False

### bias_term
- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 是否有bias，默认为True
- **规格限制**: True或者False

### axis
- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: InnerProduct的轴，当前支持1或者2
- **规格限制**: 1

## 输出

### y
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输出Tensor
- **规格限制**: 无
