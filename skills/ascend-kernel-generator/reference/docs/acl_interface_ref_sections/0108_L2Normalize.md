### L2Normalize

## 输入

**x**
- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：要进行归一化的输入张量
- **规格限制**：ND

## 属性

**axis**
- **是否必填**：非必填
- **数据类型**：listInt
- **参数解释**：指定归一化轴的列表
- **规格限制**：输入维度大小【-dim，dim-1】

**epsilon**
- **是否必填**：非必填
- **数据类型**：float
- **参数解释**：非零浮点常数
- **规格限制**：一般设为1e-4

## 输出

**y**
- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：归一化后的输出张量
- **规格限制**：ND
