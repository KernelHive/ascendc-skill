### Interp

## 输入

**x**

- **是否必填**：必填
- **数据类型**：float、float16
- **参数解释**：要进行双线性插值的输入
- **规格限制**：NCHW

## 属性

**half_pixel_centers**

- **是否必填**：否
- **数据类型**：bool
- **参数解释**：-
- **规格限制**：默认 false

**align_corners**

- **是否必填**：否
- **数据类型**：bool
- **参数解释**：-
- **规格限制**：默认 false

**height**

- **是否必填**：必填
- **数据类型**：int
- **参数解释**：输出的高
- **规格限制**：无

**width**

- **是否必填**：必填
- **数据类型**：int
- **参数解释**：输出的宽
- **规格限制**：无
