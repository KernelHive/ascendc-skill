### Clip

## 功能
将输入 Tensor 的值剪辑到指定的最小值和最大值之间，即将其限制在一个间隔内。当 `min` 大于 `max` 时，剪辑操作符将所有输入 Tensor 设置为 `max` 的值。

## 输入
- **x**：输入 Tensor，数据类型支持 float16、float、int32。
- **min**（可选）：一个 Scalar，缺少时不剪辑 x 最小值。
- **max**（可选）：一个 Scalar，缺少时不剪辑 x 最大值。

## 输出
- **y**：输出 Tensor，剪辑后的输出，数据类型和 shape 与输入一致。

## 约束与限制
不支持输入 min 和 max 同时缺少。

## 支持的 ONNX 版本
Opset v11/v12/v13/v14/v15/v16
