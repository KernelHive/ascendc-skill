### Reduction

## 输入

### x

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: -
- **规格限制**: float16

## 属性

### operation

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: default=1，含义如下：
  1. SUM
  2. ASUM
  3. SUMSQ
  4. MEAN
- **规格限制**: 支持配置，配置范围1~4

### axis

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 默认为0，表示从第几个维度开始，后面的维度全部reduction
- **规格限制**: [-rank(x), rank(x))

### coeff

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为1.0，输出的系数
- **规格限制**: 支持配置

## 输出

### y

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: -
- **规格限制**: float16
