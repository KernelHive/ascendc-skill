### Conv

## 功能
卷积操作，使用输入张量和滤波器计算输出。

## 输入
- **X**：来自上一层的输入四维或五维张量。
  - 四维输入支持数据类型：float16、float
  - 五维输入支持数据类型：float16

- **W**：权重张量。
  - 二维卷积形状：[C, M/group, kH, kW]
    - C：通道数
    - kH、kW：内核高度和宽度
    - M：特征图数量
  - 超过二维卷积形状：[C, M/group, kD, kH, kW]
  - 输出通道数 = W.shape[1] * group（形状数组索引从0开始）
  - 支持数据类型：float16

- **B**（可选）：一维输入张量。
  - 形状：[M]
  - 支持数据类型：float16

## 属性
- **auto_pad**（可选）：
  - X为NCHW时支持：NOTSET、SAME_UPPER、SAME_LOWER、VALID
  - X为NCDHW时支持：NOTSET、VALID
  - 默认值：NOTSET（使用显式填充）
  - SAME_UPPER/SAME_LOWER：填充输入使每个轴的输出尺寸与输入相同
    - 填充在两侧均匀分配（偶数时相等，奇数时不等）
    - 奇数填充时：
      - SAME_UPPER：在末尾添加额外填充
      - SAME_LOWER：在开头添加额外填充

- **dilations**：扩张率
  - 四维输入：2个整数列表
  - 五维输入：5个整数列表
  - 指定扩张卷积的扩张率
  - H和W维度取值范围：[1, 255]
  - 默认值：各空间轴dilation为1

- **group**：
  - 数据类型：int32
  - 必须设置为1
  - 输入通道到输出通道的阻塞连接数
  - 输入通道和输出通道都必须被group整除

- **pads**：填充
  - 四维输入：4个整数列表
  - 五维输入：6个整数列表
  - 指定顶部、底部、左侧、右侧填充
  - 取值范围：[0, 255]

- **strides**：步长
  - 四维输入：4个整数列表
  - 五维输入：5个整数列表
  - 指定沿高度H和宽度W的卷积步长
  - H和W维度取值范围：[1, 63]
  - 默认值：N和C尺寸设置为1

## 输出
- **Y**：卷积结果的输出数据张量
  - 输出维度是内核大小、步幅大小和pad长度的函数

## 约束
- X、W维度取值范围：[1, 4096]
- W、H维度和W维度取值范围：[1, 255]
- 不支持atc工具`--precision_mode=must_keep_origin_dtype`参数时，输入类型为float和float64

## 支持的 ONNX 版本
Opset v9/v10/v11/v12/v13/v14/v15/v16/v17/v18
