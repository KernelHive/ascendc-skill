### BatchedMatMul

## 输入

### x1

- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: 输入x1，维数必须大于2而小于等于8，如果adj_x1是False，表示[batch,…,M,K]；如果adj_x1是True，表示[batch,…,K,M]，其中batch,…, 需要与x2中的一致
- **规格限制**: 无

### x2

- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: 输入x2，维数必须大于2而小于等于8，如果adj_x2是False，shape表示为[batch,…,K,N]；如果adj_x2是True，shape表示为[batch,…,N,K]，其中batch,…, 需要与x1中的一致
- **规格限制**: 无

## 属性

### adj_x1

- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: x1是否转置，默认值为False
- **规格限制**: 无

### adj_x2

- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: x2是否转置，默认值为False
- **规格限制**: 无

## 输出

### y

- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: 输出y=x1*x2，shape为[batch,…,M,N]
- **规格限制**: 无
