### ELU

## 输入

### x

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 激活函数，融合了sigmoid和ReLU
- **计算公式**:
  ```python
  if x > 0:
      y = x
  else:
      y = alpha * (exp(x) - 1)
  ```
- **规格限制**: float16

## 属性

### alpha

- **是否必填**: 非必填
- **数据类型**: float，默认值为1
- **参数解释**: 激活函数，融合了sigmoid和ReLU
- **计算公式**:
  ```python
  if x > 0:
      y = x
  else:
      y = alpha * (exp(x) - 1)
  ```
- **规格限制**: float16
