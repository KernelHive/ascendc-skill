### Gemm

## 功能
计算 alpha 乘以 A 与 B 的乘积，再与 beta 和 input C 的乘积求和。

## 输入
- **A**：输入二维矩阵 Tensor
  - 数据类型：float16、float
  - 数据类型需要与 C、B 构成互相推导关系
  - shape（或转置后 shape）需要满足与 B 相乘条件
  - 数据格式支持 ND
- **B**：输入二维矩阵 Tensor
  - 数据类型：float16、float
  - 数据类型需要与 C、A 构成互相推导关系
  - shape（或转置后 shape）需要满足与 A 相乘条件
  - 数据格式支持 ND
- **C（可选）**：偏差（该参数暂不支持）

## 属性
- **transA**：
  - 数据类型：bool
  - 是否对 A 进行转置
  - 默认值：0（0 表示不转置，非 0 表示转置）
- **transB**：
  - 数据类型：bool
  - 是否对 B 进行转置
  - 默认值：0（0 表示不转置，非 0 表示转置）
- **alpha**：
  - 数据类型：float
  - 该参数暂不支持
- **beta**：
  - 数据类型：float
  - 该参数暂不支持

## 输出
- **Y**：输出二维矩阵 Tensor
  - shape 需要与 A@B 计算结果一致
  - 数据类型需要与 C 构成互相推导关系
  - 数据类型：float16、float

## 约束
v8/v9/v10 版本不支持 atc 工具参数 `--precision_mode=must_keep_origin_dtype` 时的 float 类型输入。

## 支持的 ONNX 版本
Opset v8/v9/v10/v11/v12/v13/v14/v15/v16/v17/v18
