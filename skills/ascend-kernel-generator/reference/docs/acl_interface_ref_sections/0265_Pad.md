### Pad

## 功能
对输入 Tensor 做填充。

## 输入
- **x**：输入 Tensor，数据类型支持 float16、float、int32。
- **pads**：输入 Tensor，数据类型支持 int32、int64。
- **constant_value**（可选）：输入 Tensor，默认值为 0、空字符串或 False。如果选择的模式为 `constant`，则要使用的标量值。

## 属性
- **mode**：string 类型，支持模式有：constant、reflect、edge。

## 输出
- **y**：输出 Tensor，数据类型和输入 x 一致。

## 约束与限制
当 mode 值为 constant 时，目前仅支持 constant_value=0。

## 支持的 ONNX 版本
Opset v11/v13/v14/v15/v16/v17/v18。
