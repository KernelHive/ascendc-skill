### MVN

## 输入

### x

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输入Tensor
- **规格限制**: float16

## 属性

### normalize_variance

- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 设为false时只减均值，默认为true
- **规格限制**: True或者False

### across_channels

- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 设为true时就是CHW视为一个向量，默认为false
- **规格限制**: True或者False

### eps

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: Epsilon为了防止除零，默认值1e-9
- **规格限制**: 可以配置

## 输出

### y

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: y=(x-mean(x))/std(x)
- **规格限制**: float16
