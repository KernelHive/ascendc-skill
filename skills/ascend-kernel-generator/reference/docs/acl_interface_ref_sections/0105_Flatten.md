### Flatten

## 输入

### x

- **是否必填**: 必填
- **数据类型**: float16、float32、int8、uint8、int16、uint16、int32、uint32、int64、uint64
- **参数解释**: 输入Tensor
- **规格限制**: 无

## 属性

### axis

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 需要flatten的开始轴，可以为负数，如果为-1表示最后一个轴，默认值为1
- **规格限制**: [-rank(x), rank(x))

### end_axis

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 需要flatten的结束轴，可以为负数，如果为-1表示最后一个轴，默认值1
- **规格限制**: [0, rank(x))

## 输出

### y

- **是否必填**: 必填
- **数据类型**: float16、float32、int8、uint8、int16、uint16、int32、uint32、int64、uint64
- **参数解释**: 输出Tensor
- **规格限制**: 无
