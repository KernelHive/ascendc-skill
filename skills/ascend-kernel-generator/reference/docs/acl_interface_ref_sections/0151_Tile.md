### Tile

## 输入

**x**:

- **是否必填**: 必填
- **数据类型**: float16、float32、int8、uint8、int16、uint16、int32、uint32、int64、uint64
- **参数解释**: 输入Tensor
- **规格限制**: 无

## 属性

**axis**:

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 指定扩展的轴，默认为1
- **规格限制**: 无

**tiles**:

- **是否必填**: 必填
- **数据类型**: int
- **参数解释**: 扩展的倍数，比如x shape [n,c,h,w]，axis=2，tiles=3，那么y shape [n,c,3*h,w]
- **规格限制**: 无

## 输出

**y**:

- **是否必填**: 必填
- **数据类型**: float16、float32、int8、uint8、int16、uint16、int32、uint32、int64、uint64
- **参数解释**: 输出Tensor
- **规格限制**: 无
