### Yolo

## 输入

### x

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: [batch, boxes * (coords + 1 + classes), height, width]
- **规格限制**: float16

### 属性

#### boxes

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 每个 grid 的 anchor box 的数量，默认值 3
- **规格限制**: 无

#### coords

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: coords 的数量，固定为 4，表示 x、y、h、w
- **规格限制**: 4

#### classes

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 类别数，默认为 80，不能超过 1024
- **规格限制**: 最大 1024

#### yolo_version

- **是否必填**: 非必填
- **数据类型**: string
- **参数解释**: “V2” 或者 “V3”，表示 YOLO V2 和 V3 的版本；分别对应于 detection_layer 和 Yolo_layer，默认值 "V3"
- **规格限制**: V2 或者 V3

#### softmax

- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 在 YOLO V2 和 V3 中，表示是否做 softmax 操作，默认值 False
- **规格限制**: True 或者 False

#### softmaxtree

- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: softmaxtree 固定为 False，接口保留（darknet 中有定义）
- **规格限制**: False

#### background

- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 在 YOLO V2 和 V3 中，与 “softmax” 参数配合，用于确定对 b、classes 等做的操作类型，默认值 False
- **规格限制**: True 或者 False

## 输出

### coord_data

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: [batch, boxes * coords, ceil(height * width * 2 + 32, 32) / 2]，表示预测的 coords
- **规格限制**: float16

### obj_data

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: [batch, ceil(boxes * height * width * 2 + 32, 32) / 2]，此处每个 anchor 的 obj 的数值只有 1 个
- **规格限制**: float16

### classes_data

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: [batch, classes, ceil(boxes * height * width * 2 + 32, 32) / 2]，为便于内部计算已将每个 anchor 的 score 向 16 取整
- **规格限制**: float16
