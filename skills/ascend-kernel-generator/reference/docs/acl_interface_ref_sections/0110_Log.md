### Log

## 输入

### x

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 指数运算，计算公式：
  - 如果 base > 0: y = log_base(shift + scale * x)
  - 如果 base == -1: y = log_e(shift + scale * x)
- **规格限制**: float16

> 算子原型 TensorFlow 和 Caffe 原型不一致，需要重新定义。

## 属性

### base

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为 -1.0
- **规格限制**: base > 0 或者 -1

### scale

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为 1.0
- **规格限制**: 无

### shift

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为 0.0
- **规格限制**: 无

## 输出

### y

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 指数运算，计算公式：
  - 如果 base > 0: y = log_base(shift + scale * x)
  - 如果 base == -1: y = log_e(shift + scale * x)
- **规格限制**: float16
