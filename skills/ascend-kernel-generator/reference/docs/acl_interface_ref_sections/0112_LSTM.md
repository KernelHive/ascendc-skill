### LSTM

## 输入参数

### x
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: 时间变化数据x，[T,N,input_size]
- **规格限制**: T最大支持256

### cont
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 序列连续性标记，[T,N]
- **规格限制**: T最大支持256

### w_x
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: x权重，维度为[input_size, 4×num_output]
- **规格限制**: 无

### bias
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 偏置，维度为[4×num_output]
- **规格限制**: 无

### w_h
- **是否必填**: 必填
- **数据类型**: float16
- **参数解释**: h权重，维度为[4×num_output, num_output]
- **规格限制**: 无

### x_static
- **是否必填**: 非必填
- **数据类型**: float16
- **参数解释**: 时间不变的静态数据x_static[N,input_size]；接口当前保留
- **规格限制**: 当传入static输入时，要求inputFM和HiddenSize都必须是16的倍数

### h_0
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: expose_hidden==True时输入，初始隐藏状态[1,N,num_output]
- **规格限制**: 无

### c_0
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: expose_hidden==True时输入，初始cell状态 [1,N,num_output]
- **规格限制**: 无

### w_x_static
- **是否必填**: 非必填
- **数据类型**: float16
- **参数解释**: x_static权重，维度为[4×num_output,input_size]; 接口当前保留
- **规格限制**: 无

## 属性

### num_output
- **是否必填**: 必填
- **数据类型**: uint
- **参数解释**: 输出特征数，必须大于0
- **规格限制**: 必须大于0

### expose_hidden
- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 是否有初始隐藏状态和cell初始状态，默认为False
- **规格限制**: 无

## 输出参数

### h
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输出的Tensor (T×N×num_output)
- **规格限制**: 无

### h_t
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: 最后timestep的隐藏状态 (1×N×num_output)
- **规格限制**: 无

### c_t
- **是否必填**: 非必填
- **数据类型**: float16、float32
- **参数解释**: 最后timestep的cell状态 (1×N×num_output)
- **规格限制**: 无
