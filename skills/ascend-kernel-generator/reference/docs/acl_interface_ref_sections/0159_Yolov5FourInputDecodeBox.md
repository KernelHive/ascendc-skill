### Yolov5FourInputDecodeBox

## 输入

### x1
- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：64倍下采样的feature_map
- **规格限制**：NCHW

### x2
- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：32倍下采样的feature_map
- **规格限制**：NCHW

### x3
- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：16倍下采样的feature_map
- **规格限制**：NCHW

### x4
- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：8倍下采样的feature_map
- **规格限制**：NCHW

## 属性

### bias_0
- **是否必填**：必填
- **数据类型**：float
- **参数解释**：第一个输入feature_map对应的anchor数据
- **规格限制**：6个值

### bias_1
- **是否必填**：必填
- **数据类型**：float
- **参数解释**：第二个输入feature_map对应的anchor数据
- **规格限制**：6个值

### bias_2
- **是否必填**：必填
- **数据类型**：float
- **参数解释**：第三个输入feature_map对应的anchor数据
- **规格限制**：6个值

### bias_3
- **是否必填**：必填
- **数据类型**：float
- **参数解释**：第四个输入feature_map对应的anchor数据
- **规格限制**：6个值

### ori_size_h
- **是否必填**：必填
- **数据类型**：int
- **参数解释**：原图高

### ori_size_w
- **是否必填**：必填
- **数据类型**：int
- **参数解释**：原图宽

### anchor
- **是否必填**：必填
- **数据类型**：int
- **参数解释**：anchor数
- **规格限制**：只支持等于3

### class_num
- **是否必填**：必填
- **数据类型**：int
- **参数解释**：类别数
- **规格限制**：1到80类

### thresh_front
- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：置信度和类别得分阈值
- **规格限制**：0.0-1.0之间

## 输出

### y
- **是否必填**：必填
- **数据类型**：float16
- **参数解释**：解析feature_map坐标和置信度类别得分的输出结果
- **规格限制**：NCHW
