### Mod

## 功能
执行逐元素取模（支持numpy风格的广播）。余数的符号与除数的符号相同。

## 输入
- **x**：输入Tensor，数据类型支持float16、float、int32、int8、uint8
- **divisor**：输入Tensor，数据类型支持float16、float、int32、int8、uint8

## 输出
- **y**：输出Tensor，数据类型支持float16、float、int32、int8、uint8

## 约束与限制
无

## 支持的 ONNX 版本
Opset v10/v11/v12/v13/v14/v15/v16/v17/v18
