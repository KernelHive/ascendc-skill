### BitwiseOr

## 功能
计算 Tensor A 中每个元素与 Tensor B 中对应位置的元素的按位或。

## 输入
- **A**：输入 Tensor，数据类型支持 int8、int16、int32、int64、uint8、uint16、uint32、uint64
- **B**：输入 Tensor，数据类型支持 int8、int16、int32、int64、uint8、uint16、uint32、uint64

## 输出
- **C**：输出 Tensor，数据类型支持 int8、int16、int32、int64、uint8、uint16、uint32、uint64

## 约束与限制
无

## 支持的 ONNX 版本
Opset v18
