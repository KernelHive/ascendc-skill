### ROIAlign

## 输入

### features
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输入Tensor
- **规格限制**: 无

### rois
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输入数据为[N,5]或者[N,8]，其中的5表示batchID、x1、y1、x2、y2；[N,8]中的8也是前5个数据有效，后面自动补充3位对齐
- **规格限制**: 无

### rois_n
- **是否必填**: 非必填
- **数据类型**: int32
- **参数解释**: Tensor的Shape为[batch,8]，其中8表示8列数中只有第一列有效（为满足性能加速，进行数据对齐，后面7列数字是补齐的无效数据），表示每个batch实际的rois数量。  
  例如，Shape为[4,8]，其中只有首列表示有效数据，存放每个batch的rois数量：
  ```
  [0,0]...[0,7]
  [1,0]...[1,7]
  [2,0]...[2,7]
  [3,0]...[3,7]
  ```
- **规格限制**: 该参数可选

## 属性

### spatial_scale
- **是否必填**: 必填
- **数据类型**: float
- **参数解释**: rois相对features的缩放比例
- **规格限制**: 无

### pooled_h
- **是否必填**: 必填
- **数据类型**: int
- **参数解释**: 输出y的height
- **规格限制**: 无

### pooled_w
- **是否必填**: 必填
- **数据类型**: int
- **参数解释**: 输出y的width
- **规格限制**: 无

### sampling_ratio
- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 用于确定每个bin中采样几个点求均值
- **规格限制**: 2

## 输出

### output
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 无
- **规格限制**: 无
