### ReLU/LeakyReLU/RReLU

## 输入

**参数名**: x

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输入
- **规格限制**: 无

## 属性

**参数名**: negative_slope

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 负值的斜率，默认为0
- **规格限制**: 无

## 输出

**参数名**: y

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输出 y = max(0, x) + negative_slope * min(0, x)
- **规格限制**: 无
