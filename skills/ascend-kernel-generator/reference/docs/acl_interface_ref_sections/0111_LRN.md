### LRN

## 输入

### x

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: normalization的方法，计算公式: `power(k + (alpha / n) * sum(square(x)), beta)`
- **规格限制**: float16

## 属性

### local_size

- **是否必填**: 非必填
- **数据类型**: uint
- **参数解释**: 默认为5，必须是奇数（LRN only supports odd values for local_size），local_size不大于255
- **规格限制**: 必须是奇数，且不大于255

### alpha

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为1.0
- **规格限制**: 无

### beta

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为0.75
- **规格限制**: 无

### norm_region

- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**:
  - 0: ACROSS_CHANNELS
  - 1: WITHIN_CHANNEL
  - 默认为0，只支持ACROSS_CHANNELS
- **规格限制**: 只支持0

### k

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认为1.0
- **规格限制**: 无

## 输出

### y

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: normalization的方法，计算公式: `power(k + (alpha / n) * sum(square(x)), beta)`
- **规格限制**: float16
