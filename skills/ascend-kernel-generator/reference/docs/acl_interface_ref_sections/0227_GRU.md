### GRU

## 功能
计算单层GRU，通常通过一些自定义实现来支持此运算符。

## 输入

### x
输入Tensor，将输入序列打包成一个三维张量。

- **Shape**: `[seq_length, batch_size, input_size]`
- **数据类型**: float16

### w
输入Tensor，gate的权重，（如果是双向的）沿维度0方向串联。

- **Shape**: `[num_directions, 3*hidden_size, input_size]`
- **数据类型**: float16

### r
输入Tensor，递归权重Tensor，（如果是双向的）沿维度0方向串联。

- **Shape**: `[num_directions, 3*hidden_size, hidden_size]`
- **数据类型**: float16

### b（可选）
输入Tensor，gate的偏移Tensor，（如果是双向的）沿维度0方向串联。

- **Shape**: `[num_directions, 6*hidden_size]`
- **默认值**: 0（如未选中）
- **数据类型**: float16

### sequence_lens（可选）
输入Tensor，用于指定某批次中序列的长度。

- **Shape**: `[batch_size]`
- **默认值**: 假定所有序列都有length（如未选中）
- **数据类型**: int32

### initial_h（可选）
输入Tensor，hidden的初始值。

- **Shape**: `[num_directions, batch_size, hidden_size]`
- **默认值**: 0（如未选中）
- **数据类型**: float16

## 属性

### activation_alpha
激活函数使用的可选缩放值。这些值按激活函数的顺序使用，例如LSTM中的(f, g, h)。

- **数据类型**: floats
- **默认值**: 与相应的ONNX一样，而对于LeakyRelu，默认alpha为0.01

### activation_beta
激活函数使用的可选缩放值。这些值按激活函数的顺序使用，例如LSTM中的(f, g, h)。

- **数据类型**: floats
- **默认值**: 与相应的ONNX一样

### activations
用于更新、重置和隐藏gates（如果是双向）激活函数列表。激活函数必须是上述指定的激活函数之一。

- **数据类型**: strings

### clip
单元clip阈值，将Tensor元素限制在具体单位内。

- **数据类型**: float

### direction
指定RNN是正向、反向还是双向。

- **数据类型**: string
- **默认值**: forward
- **可选值**: forward（默认）、reverse、bidirectional

### hidden_size
隐藏层神经元数量。

- **数据类型**: int

### linear_before_reset
计算隐藏gates的输出时，在乘以复位门的输出之前应用线性变换。

- **数据类型**: int
- **默认值**: 0

## 输出

### y
输出Tensor，连接了 hidden 的所有中间输出值。

- **Shape**: `[seq_length, num_directions, batch_size, hidden_size]`
- **数据类型**: float16

### y_h
输出Tensor，隐藏的最后一个输出值。

- **Shape**: `[num_directions, batch_size, hidden_size]`
- **数据类型**: float16

## 约束与限制
无。

## 支持的 ONNX 版本
Opset v8/v9/v10/v11/v12/v13/v14/v15/v16/v17/v18
