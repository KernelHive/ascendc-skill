### Slice

## 输入

**x**

- **是否必填**: 必填
- **数据类型**: float16、float32、int8、uint8、int16、uint16、int32、uint32、int64、uint64
- **参数解释**: 沿某个维度切割输入的 blob，输出多个切割的 blob
- **规格限制**: 无

## 属性

**slice_dim**

- **是否必填**: -
- **数据类型**: uint
- **参数解释**: 默认为 1，跟 axis 含义相同
- **规格限制**: 支持配置 [1, rank(x))，遵循 Caffe 规则

**slice_point**

- **是否必填**: -
- **数据类型**: uint
- **参数解释**: 表示输出的各个 blob 的切割点，不定义的则均分
- **规格限制**: 支持，切割点数小于 32

**axis**

- **是否必填**: -
- **数据类型**: int
- **参数解释**: 默认为 1，表示切块的维度
- **规格限制**: [-rank(x), rank(x))，遵循 Caffe 规则

## 输出

**y**

- **是否必填**: 必填
- **数据类型**: float16、float32、int8、uint8、int16、uint16、int32、uint32、int64、uint64
- **参数解释**: 沿某个维度切割输入的 blob，输出多个切割的 blob
- **规格限制**: 无
