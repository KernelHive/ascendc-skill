### Proposal

## 输入

### cls_prob
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 前景和背景的概率，[batch, num_anchors*2, height, width]，*2表示背景和前景概率（第一个是背景概率bg prob，第二个是前景概率fg probs）
- **规格限制**: 无

### bbox_delta
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 预选框位置（Preselected box position），[batch, num_anchors*4, height, width]
- **规格限制**: 无

### im_info
- **是否必填**: 非必填
- **数据类型**: -
- **参数解释**: 输入图片的大小（高和宽），默认值为 {375, 1240}
- **规格限制**: 无

## 属性

### feat_stride
- **是否必填**: 必填
- **数据类型**: float
- **参数解释**: 滑块滑动的大小，默认值 16
- **规格限制**: 无

### base_size
- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: generate_anchors 中生成基本框的大小，默认值 16
- **规格限制**: 无

### min_size
- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: box 的最小大小，默认值: {16, 16}
- **规格限制**: 无

### ratio
- **是否必填**: 非必填
- **数据类型**: ListFloat
- **参数解释**: generate_anchors 中生成框的高宽比例，默认值为 [0.5, 1, 2]
- **规格限制**: 无

### scale
- **是否必填**: 非必填
- **数据类型**: ListFloat
- **参数解释**: generate_anchors 中生成框的高宽相对于 base_size 的大小，默认值为 {8, 16, 32}
- **规格限制**: 无

### pre_nms_topn
- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: Topk 中的 K，在 float16 场景下，非昇腾 AI 处理器场景中最大为 6008，昇腾 AI 处理器场景中最大 3008；当采用 FP32 时，非昇腾 AI 处理器场景中最大为 3004，昇腾 AI 处理器场景中最大 1502（默认值 1502）
- **规格限制**: 不超过 1502

### post_nms_topn
- **是否必填**: 非必填
- **数据类型**: int
- **参数解释**: 最终的输出数量，非昇腾 AI 处理器场景中最大为 6000，昇腾 AI 处理器场景最大 3000；当采用 FP32 时，非昇腾 AI 处理器场景中最大为 3000，昇腾 AI 处理器场景中最大 1502（默认值 304）
- **规格限制**: 不超过 1502

### iou_threshold
- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 交并比（Intersection over Union）的阈值，必须是 (0,1)，默认为 (0.7)
- **规格限制**: (0,1)

### output_actual_rois_num
- **是否必填**: 非必填
- **数据类型**: bool
- **参数解释**: 是否输出 actual_rois_num，False: 不输出，默认为 False；True: 输出
- **规格限制**: 无

## 输出

### rois
- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 输出框信息，Tensor[batch, 5, N]，其中 N 是 post_nms_topn 对齐到 16 的倍数的值，5 表示（batchID, x1, y1, x2, y2），实际每个 batch 按 actual_rois_num 输出
- **规格限制**: 无

### actual_rois_num
- **是否必填**: 非必填
- **数据类型**: int32
- **参数解释**: [batch, 8]，其中 8 表示 8 列数中只有第一列有效（为满足性能加速，进行数据对齐，后面 7 列数字是补齐的无效数据），表示实际每个 batch 输出的 bbox 数量。例如，Shape 为 [4, 8]，其中只有首列表示有效数据，存放每个 batch 的 bbox 数量：
  ```
  [0,0]...[0,7]
  [1,0]...[1,7]
  [2,0]...[2,7]
  [3,0]...[3,7]
  ```
- **规格限制**: 无
