### LpNormalization

## 功能
给定一个矩阵，沿给定的轴应用 Lp 归一化。

## 输入
- **x**：输入 Tensor，数据类型支持 float16、float。

## 属性
- **axis**：数据类型为 int，应用归一化的轴，-1 表示最后一个轴。默认值为 -1。
- **p**：数据类型为 int，归一化的阶数，仅支持 1 或 2。默认值为 2。

## 输出
- **y**：输出 Tensor，数据类型和 shape 与输入一致。

## 约束与限制
无。

## 支持的 ONNX 版本
Opset v8/v9/v10/v11/v12/v13/v14/v15/v16/v17/v18
