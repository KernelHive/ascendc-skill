### Mul

## 功能
执行逐元素乘法计算（支持 NumPy 风格的广播）。

## 输入
- **A**：输入 Tensor，shape 需要与 B 满足广播关系，数据类型支持 float、float16、int8、int32。
- **B**：输入 Tensor，shape 需要与 A 满足广播关系，数据类型支持 float、float16、int8、int32。

## 输出
- **C**：输出 Tensor，shape 需要与 A 和 B 进行广播后的 shape 相等，数据类型与输入 A 保持一致。

## 约束与限制
无。

## 支持的 ONNX 版本
Opset v8/v9/v10/v11/v12/v13/v14/v15/v16/v17/v18。
