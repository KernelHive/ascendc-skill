### Power

## 输入

**x**

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 幂函数，计算公式: `y = (shift + scale * x) ^ power`
- **规格限制**: 无

## 属性

**power**

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认值 1.0
- **规格限制**: power 是有小数位的，要求底数大于 0，其它按 Caffe 定义

**scale**

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认值 1.0
- **规格限制**: 无

**shift**

- **是否必填**: 非必填
- **数据类型**: float
- **参数解释**: 默认值 1.0
- **规格限制**: 无

## 输出

**y**

- **是否必填**: 必填
- **数据类型**: float16、float32
- **参数解释**: 幂函数，计算公式: `y = (shift + scale * x) ^ power`
- **规格限制**: 无
