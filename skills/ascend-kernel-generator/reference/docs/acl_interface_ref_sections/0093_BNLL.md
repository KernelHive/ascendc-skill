### BNLL

## 输入

**x：**

- **是否必填：** 必填
- **数据类型：** float16、float32
- **参数解释：** 激活函数
- **计算公式：**

```python
if x > 0:
    y = x + log(1 + exp(-x))
else:
    y = log(1 + exp(x))
```

- **规格限制：** 无

## 输出

**y：**

- **是否必填：** 必填
- **数据类型：** float16、float32

